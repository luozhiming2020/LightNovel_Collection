小说名称：欢迎来到血吸村
本卷名称：第五卷

第一话 吸血鬼的昏睡症
台版 转自 艾丝蒂尔@轻之国度

1

「直树，你的应该差不多了吧？」

高村直树被问到这句话，是在离黄金周假期没几天的某个平日午后，地点则是教室。

「啥？」

直树听了不禁一愣。因为他完全没听懂对方的意思。

向他如此小声询问的，是一位带着男孩子气的短发少女。她的皮肤黝黑，身材修长，一看就能知道她擅长体育活动。

她的名字是利根崎智衣。既是在这治水高中就读的女高中生，也是直树的同班同学。

「你说什么差不多了？我的生日还很久啊。」

「笨蛋，我不是在说生日啦。」

既然如此，直树就想不到会是什么了。唯一掠过他脑里的，就只剩下小学六年级时，喝醉酒的父亲对他开的玩笑话。

『直树，你的应该差不多了吧？底下的毛长出来了吧？哇哈哈哈！』

（不可能吧，女孩子怎么会问这种事。）

虽然直树第一时间轻笑着这么想，但他很快又改变了想法。

（智衣平常豪迈又开朗，个性简直和男孩子差不多。而且，也说过把我当成挚友看待。和我之间的关系与其说是异性，大概和其他男生朋友一样吧。）

尽管这是天大的误会，但直树就是不明白。

（她大概是学其他男生找我开开玩笑吧。）

即使如此，为了避免猜错而带来的尴尬，直树还是小小声地问道：

「欸，智衣你是在问我关于毛的事情吗？」

「喔，对啊。」

智衣大大点头。

（什么啊，原来真的是在开我玩笑。）

直树认为应该配合对方的玩笑，所以笑着回答：

「智衣，你在说什么啊，我很早以前就长啦。一般男生在国中的时候就长齐了吧？」

「啥？」

这次却轮到智衣听不懂了。

「你在说的是下面的毛没错吧？」

刚开始智衣还是没有意会过来，呆了一会儿，但是她的脸一下子变得红通通的。

「笨、笨蛋！」

智衣一把推开直树。

「我为什么要问你那种事情啦！而且那根本不用问也知道吧。你到底在想什么啊。真是，笨蛋笨蛋笨蛋笨蛋！」

智衣用力挥舞着双臂，不过很快就回过神来。因为她发现到身边的其他同学正在注视着这里。

「我、我们只是在开玩笑而已啦，不用在意。」

她傻笑了几声蒙混过去，然后一把抓住倒在地上的直树衣领，把他整个人提起来。

然后，在直树耳朵旁小声问道：

「我在问的是你头上的毛，头发差不多该剪了吧？」

「啊。」

直到此时，直树才总算意会过来。距离前一次剪头发已经是三个礼拜前的事情了。智衣说的没有错，的确该剪了。如果继续丢着不管，班上同学或其他村民将会发现他的头发变长。

也许有些人会说，头发变长也无所谓吧，那不是很自然的事情吗？但事情就是没那么单纯。

在这村子里，头发变长是不可能的事情。

其实，这座村子里的居民全部都是吸血鬼。他们的时间在变成吸血鬼时就已经停止了。不只是头发不会变长，连身体也不会成长。

对于必须隐瞒自己还是人类之身，假装成吸血鬼在村子里生活的直树而言，头发会变长绝对是个不能说的秘密。

「放学后我去帮你剪吧？虽然我要参加社团活动，还要帮家里的店。不过如果你有麻烦的话，我是一定帮的。」

尽管听起来像是在硬塞人情，智衣还是很积极地提议。直树很感谢她的好意，但就是无法点头。

因为直树很清楚智衣的技术，他并没有忘记去年暑假被迫顶着光头度过的悲剧。

「没啦，你忙你的，我还是跟以前一样去找川澄剪吧。也是，我应该趁还没忘记的时候去找她。」

说着，直树就走向教室角落的座位。

「什么嘛，我都已经说要帮你剪了。」

智衣把头转向一边，不满地嘟起嘴。

「嘿，川澄。」

听到直树的声音，一名女学生抬起头。她是一个看起来十分文静的女孩子，外型也十足是位美少女。

她就是与直树同班的川澄季代实。虽然看起来有点冷淡，其实只是不擅长将感情显露在脸上而已。每当直树的人类身分快要穿帮时，这位女孩子总是会竭尽全力帮助他。

「想拜托你一件事情，就是我的毛。啊，我说的不是那边的毛，是再更普通一点的部位。」

直树开始把事情愈描愈黑。平常的话季代实一定会赏他一记白眼，但这次却不太一样。

季代实的眼神看起来很空洞，上半身甚至摇摇晃晃的。

即使比较迟钝一点的直树也看出她的情况不太对，担心地问道。

「川澄，你身体不舒服吗？看起来好像无精打采的。」

「没有那种事。」

季代实的声音有点虚弱。

「我只是……有点困而已。」

「你昨天有熬夜吗？」

「我……没有。睡觉的时间和平常一样，不可能是睡眠不足。」

尽管她揉了揉眼睛，但还是没能打起精神来。

「没关系，不必在意我的事情。刚才你说的我都明白了，今天放学以后我就去找你……」

季代实话只说到一半，她就全身无力地从椅子上滚落，整个人倒落在教室的地上。

「喂，川澄！振作一点！川澄！」

然·后……

「嘿咻。」

智衣把一路背过来的季代实放到保健室的床铺上。

「如果这么想睡的话，干脆翘课不就好了。你还真是死脑筋。」

替季代实盖上薄被后，智衣自言自语着。

「你就在这里好好睡吧。」

朝沉睡的季代实说完话后，智衣又转头看向自己身后的直树。

「好了，我们回教室吧。」

「可是，真的好吗？」

直树一脸不安地问道。

「是不是和保健室的老师说一声比较好？她会不会是得了什么严重的病？」

听到直树的话，智衣不禁笑了出来。

「怎么可能啊？我们可是吸血鬼，根本不可能生病啊。我已经在这村子住了快二十年，连感冒都没有得过耶。」

智衣手撑着腰，豪爽地大笑着。

（以智衣来说，就算不是吸血鬼应该也不会感冒吧？病毒看到她都要吓跑了。）

虽然直树脑袋里这么想，但他可不敢说出口。毕竟现在惹智衣生气一点好处也没有。

而且，智衣说的没有错。吸血鬼特异的身体就算受伤也会立刻治愈，不太可能会输给疾病。

实际上，治水高中的保健室里根本就没有保健老师。顶多只有属于夜行性生物的吸血鬼学生们为本性所苦，白天感觉不太舒服时会过来休息一下而已。

「就是啊，吸血鬼根本不会生病嘛。」

「是啊，没错。不过也有些人会得相思病之类的啦，例如我。」

智衣有点不好意思，抓了抓鼻尖后，又挥舞着双手。

「开玩笑啦。总而言之，季代实她没事。大概是为了制作戏剧社的戏服才熬夜之类的吧，只要睡到放学就会恢复精神了。幸好下一堂课是小平老师的英文课，跟她说一声就好了。」

智衣很快就走出保健室。直树不知道该说什么，就只是低头看着躺在床上的季代实。

吸血鬼不会生病。尽管脑袋里能够理解智衣所说的这句话，但直树就是有不好的预感。

「我想太多了吧。吸血鬼根本就不会生病，她只是因为疲倦才睡着的。」

他对自己自言自语着。

「我先走啰，你好好休息吧。」

直树轻声地朝季代实说话，然后把床铺周围的布帘拉上，之后就离开了保健室。

没想到，直树的预感居然在几个小时后成真了。

2

下午的课结束后，时间来到放学。

把书包背在背后并离开教室的直树，直接前往了保健室。

直树原本以为季代实醒过来后会回到教室，但是一直等到放学都没有看见她。考量到直接回家不太妥当，直树决定到保健室看看情况。

「我进来了。」

直树打开门，并且踏入保健室。室内一片寂静。

里头的布帘还是拉上的，看来季代实还没有醒来。虽然直树不想打扰她，但现在已经到放学时间，作为一个朋友实在应该叫醒她。

「川澄，醒来了吗？」

直树从布帘外问了一声，但是没有得到回应。

「川澄，你在里头吧？已经放学啰。差不多该起床回家了吧？回家应该比较好休息吧？」

尽管稍稍提高了音量，但还是没有回应。

直树感到有些担心，只好把手放到布帘上。

「我要拉开啰～」

发出一阵喀喀的摩擦声后，直树稍稍拉开了布帘。季代实就和几个小时前没两样，依旧是闭着双眼躺在床铺上。

「现在已经放学了耶……」

直树边说着边朝床铺靠近，但季代实依然没有醒来的迹象。

他无奈地叹了口气。

到校舍关闭之前还有两个小时左右。反正最后应该会有老师来巡视，就让她睡到那时候吧。

原本直树打算就这样离去，不过他却开始看着季代实的睡脸。因为他不禁看傻了眼。

（虽然现在才说这种话好像怪怪的，不过川澄还真可爱，每个月都有这么可爱的女孩子帮自己剪头发，我应该算是很幸运的人吧？）

不过，他又用力摇头。

「我现在可是住在满是吸血鬼的危险环境下，怎么称得上是幸运。」

直树苦笑着，然后又朝躺在床上的季代实说道：

「再见啰，川澄。我先回家了。头发下次再剪好了，反正就快要放连假了。」

还是没有回应。

「这种情况，一般会形容成『睡得像是死人一样』吧。」

带着自言自语，直树离开了床铺旁。他把布帘拉上，不过马上又拉开，并且露出十分凝重的表情。

因为他听到自己说的『睡得像是死人一样』这句话后，居然开始担心了起来。

（川澄她，再怎么说也太没反应了吧？）

他走向季代实身旁。

「川澄，喂，川澄。」

直树发出比刚才大的声音。如果她的眼睑有稍微动了一下，或是有发出任何声音，那就没什么好担心了。但季代实就是没有任何反应。

直树试着把耳朵凑到季代实的嘴旁。但他很快就变了脸色。因为他完全没听到该听到的声音。

没错，就是呼吸。

「！？」

脸色大变的直树马上掀开被子，并且握住季代实的手，但得到的却是冷冰冰的感觉。就算把手指放到她的手腕上，也完全感觉不到脉搏。

「怎、怎么会……川澄她、死死死、死掉了？」

直树立刻陷入慌乱。

（对了！这种时候要赶快作CPR！要作人工呼吸！）

直树急忙把被子掀到一旁，并且把手移到季代实的胸前准备作CPR，但……

「学长你想做什么——！」

伴随着这道叫声，一个书包划破空气飞来。不只打中直树的侧脑，甚至把他击飞出去。

晕头转向的直树抬起脸来，看到的是一名保持着丢掷书包姿势的娇小女学生。闪亮的大眼睛及头发后方的大蝴蝶结很令人印象深刻。

「绘理子。」

她就是治水高中一年级学生鸠羽绘理子，正满脸通红地瞪着直树。那毫无疑问是充满愤怒的表情。

在她身后还站着露出同样表情的智衣。她似乎是在社团活动途中就跑了出来，身上还穿着运动服。

「你们两个怎么会在这里？」

「因为我想到季代实还在睡，所以才来叫她。结果半路上碰到了绘理子，两个人才一起过来……不对，这些事怎样都好！」

智衣大吼着。

「直树，我真是错看你了！怎么可以趁季代实睡着的时候，偷、偷摸她的胸部！」

「高村学长！龌龊！色狼！变态！下流！你是地球上所有女性的敌人！！」

这就是所谓的有理说不清吧。

而且，两人甚至因为怒气而释放了吸血鬼的力量。她们双眼染红，嘴角露出尖锐的虎牙，指甲也变得又利又长。

再这样下去，她们就要扑向直树了。

「你们两个都误会了。我不是要做什么下流的事情！只是要替她作CPR而已！」

直树一脸慌张，提高了音量。

「大事不妙啊！川澄她没有呼吸了！而且她的手好冰冷，也没有脉搏！」

尽管直树激动地向两人说明，但她们的脸上却挂着不解的表情。

原本还以为她们是因为事情太过突然而反应不过来，但并非如此。

降低怒气并且解除吸血鬼型态的两人，转头互望着。

「难道说，直树还不知道吗？」

「看来好像是那样呢，不然怎么会这么大惊小怪呢。」

接着，两人又一起把脸转向直树。

「直树，你不必担心太多，那没什么大不了的。」

「就是呀。这样很正常的，对包括绘理在内的吸血鬼来说。」

「咦？」

绘理子开始向有听没懂的直树说明。

「绘理熟睡的时候也会像那样。不只呼吸会停工、身体会变冷，连脉搏也会消失呢。」

「有闭眼睛的还算正常吧。我家老爸睡觉的时候连眼睛都睁着，看起来有够恐怖。」

回想起父亲的睡脸，智衣作出苦瓜脸。

「所以说，川澄她没事？不是死掉了？」

心情一放松，直树感觉自己全身都虚脱了，他整个人坐到地上。

「吓、吓死我了，反而是我的心脏差点就停了啊。」

他瞪着躺在床上的季代实，把抱怨说出口。

（话说回来，原来吸血鬼熟睡的时候会停止呼吸、身体变冷，连脉搏也会消失啊。）

直树想起以前看过的吸血鬼电影。电影主角在白天里打开吸血鬼的棺材，发现里头躺的吸血鬼睁着双眼，而且一动也不动。看来真正的吸血鬼也是这样。

他不禁感到害怕，打了个冷颤。

「不过好险啊，简直是千钧一发。」

「就是说呀，如果再晚一步，高村学长就要碰到川澄学姊的胸部了。要是继续发展成人工呼吸就更糟糕了。那样不就是接吻了吗。」

智衣与绘理子朝彼此点头。

「好——啦，那就叫醒季代实后再回社团吧。」

说着，智衣朝床铺旁走去。大概是因为身为同性，她毫不客气地摇起季代实的肩膀。

「季代实，已经放学啦。该起床罗！」

没有醒来的迹象。智衣继续摇着季代实的身体。

「喂，季代实，快起床啊。要是直树又大惊小怪地想对你做CPR怎么办？」

但季代实还是没有醒过来。想必智衣也觉得奇怪了，她开始拍打季代实的脸颊。

「喂，季代实！醒醒！快醒过来啊！！」

「利根崎学姊，不可以那么粗鲁啦！」

绘理子急忙抓住智衣的手。

「奇怪了，为什么她就是不醒过来？就算吸血鬼再怎么熟睡，这样也该醒过来啦？」

智衣不解地皱着眉头。此时，一道声音出现在保健室内。

「你们几个到底在吵闹什么！」

有着略微卷翘的蓬松长发、锐利的眼神、全身上下散发出威严气质，一位亮丽的美少女。

进入保健室的人物，就是这么一名女学生。

「红华！」

「神乐学姊！」

智衣与绘理子同时喊出声。

神乐红华，她就是治水高中的学生会长。打从本村变成吸血鬼之村开始，她就已经居住于此，因此身为吸血鬼的年资当然很长。

「还以为是谁呢，原来是你们呀。」

红华没好气地摇摇头。

「又不是小学生，在保健室里保持安静应该是常识才对吧。尤其是智衣同学，光是在走廊上就听得见你的声音了。要知道你的声音本来就够吵了哟。」

平常的话，智衣肯定不会无视她的挑衅，但这次她忍住了。

「红华，季代实的情况不太对。」

「你说川澄同学？」

「是啊，怎么样都叫不醒她！」

「她不是在熟睡吗？例如昨天熬了夜之类的。」

红华毫无紧张感地走近床铺。

「川澄同学，起来了。保健室不是让你午睡的地方哟。更何况现在已经放学了。」

红华朝她喊话。如果这样就能叫醒她，那当然是一点问题也没有。

「川澄……同学？」

红华眯起眼睛，表情上开始出现若干紧张。

下一秒钟，她已经揪住季代实的胸口，还抬起右手。

啪！啪！

两记巴掌就这么毫不客气地甩在季代实脸颊上。那两记巴掌的力道之强，刚才智衣的巴掌反而显得小家子气。

然而，这并不能阻止季代实的昏睡。季代实就和原来一样，依旧保持着沉睡。就连眉头都没有皱一下。

神情严肃的红华，以细微的声音说出一句话：

「……吸血鬼的昏睡症。」

「咦，会长？川澄果然是生病了吗？」

听到红华的话语，直树急忙询问。

「什、什么事也没有。」

不知为何，红华回答得很大声，像是在掩饰什么。

「川澄同学只是在熟睡而已，她应该是累坏了吧。这里交给我来照顾，你们先离开吧。快快，智衣同学，社团的成员都在等你回去。智衣同学可是田径社重要的一员，怎么可以不在呢？」

红华不自然地抬举了智衣，然后将三人推出房间。

「那么再会了。」

说完，红华就把保健室的门用力拉上。

被关在门外的三人不太能接受这样的作法。

「欸，红华那态度绝对很可疑吧？」

直树和绘理子都点头同意智衣的意见。

「吸血鬼的昏睡症，记得会长刚才是这么说的。」

直树回想起红华刚不小心说出口的话。

「可是，绘理从来没有听说过吸血鬼还会生病。」

「我也没有啊，这简直是前所未闻吧。」

大伙儿烦恼了一阵子，但当然没有想到答案。

或许是无法忍受凝重的气氛，智衣激动地说道：

「算了，再烦恼下去也没有用。不过就算去逼问红华她也不会回答吧。既然这样就只剩一个方法，去找别人问。」

「请问学姊说的别人是谁呀？」

「我想想看。」

似乎没有头绪，智衣环起双臂思考着。

「还是去问绫菜？她们两个平常都在学生会，也许知道些什么。」

「真的吗？」

绘理子抱持怀疑地问。

「泽田学姊身为吸血鬼的经历比利根崎学姊还要短哟，现在应该找吸血鬼当得和神乐学姊一样久的人问才对吧？」

「说到当吸血鬼和会长一样久的人。」

直树寻找自己的记忆，然后把该位人物的名字说出来。

「应该还是平山老师吧。」

「好！既然这样就马上行动吧！我们现在就去找小平吧！」

智衣带头向前走去，看来她根本没打算回社团活动。

「红华那家伙，如果知道些什么就应该告诉我们啊。我们可是一起保护直树秘密的同伴呢。」

在这个满是吸血鬼的村子里，直树之所以能够维持人类之身，是因为红华、智衣、季代实、绘理子，以及刚才略提到的绫菜这五名少女所作的一种尝试。之后，五人便立了一份协定，在让直树心甘情愿选择让谁吸血以前，其他人都不可以觊觎他的血液。而且她们十分努力不让其他村民得知直树的秘密。

也因为如此，智衣更不能接受红华的态度。

「随便她啦，不告诉我的话就自己去查好了。」

智衣像个闹别扭的孩子嘟起嘴唇走着，直树及绘理子则急忙追在她的身后。

3

一名女教师正行走在治水高中的走廊上。脸蛋可爱的她个头娇小，如果穿着制服的话，可能会被当成是高中生吧。

她的名字是平山里佳，是治水高中的英文教师。

不用说她当然是吸血鬼，而且还是本村成为吸血鬼之村时就在的老成员。

「小平老师！」

突然听到背后有人喊自己的里佳，停下脚步并且回头看。她看见了三名学生朝自己这里跑来。

「哎呀，利根崎同学，你们有什么事吗？」

「其实我们有些问题想问小平老师。总之请你先跟我们来一下。」

智衣勾着里佳的手臂，然后拉着她走。一直来到较为偏僻的校舍间通道角落。这里平常没什么人，是个安静的地方。

「居然把我带到这里，难道是和高村同学有关的事情吗？」

里佳露出调皮的表情小声问道。

其实里佳知道直树还是人类，也明白他目前的状况。不过，至少她并不会泄漏秘密，或者要求分一点血。

「还是说，你们是来告诉我说他已经变成吸血鬼了？」

「不、不是那样的。我现在还是人类，也没有想让任何人咬。」

看到直树慌张地摇头，里佳遗憾地叹了口气。

「都已经被这么多充满魅力的女孩子们包围，居然还没办法下定决心。高村同学，太优柔寡断的男孩子总有一天会被抛弃哟。如果对于当个吸血鬼还有不安，你可以放心。我家老公几天就习惯了，现在过得很好呢。」

她的先生在前不久还是个人类。他是在结婚典礼当天让里佳吸血，正式成为一名吸血鬼。直树也作为宾客当场目睹了过程。

「就是啊，直树。」

智衣在里佳与直树的谈话中插了话。

「你也差不多该答应了吧。别担心，我保证尽量不咬痛你。」

「请稍等一下，为什么利根崎学姊已经准备要吸血了！？高村学长的血是属于绘理的。」

绘理子不落人后地主张着。

「那是什么时候决定的？」

「当然是高村学长和绘理认识的时候就决定了。」

「你别胡说八道了！」

「胡说八道的应该是利根崎学姊才对吧！」

两人互瞪着，视线几乎要爆出火花来。

由于她们看起来就快要变成吸血鬼开始互抓，直树连忙介入两人的争吵。

「冷静一点啊！我们是来问平山老师事情的吧？」

「啊，说得也是呢。」

「都是因为绘理子乱说话害的。」

回想起原本的目的后，两人纷纷把视线转向里佳。

「小平老师，我们想问你一件事情。」

起了这样的开头后，智衣把病名说出口。

「所谓吸血鬼的昏睡症，到底是什么样的病？」

「吸血鬼的昏睡症……？」

里佳疑惑地侧着头。

「好像是吸血鬼会得的病……」

「对不起，我没有听说过。」

就连她这个最早的成员都不知道，想必再去问任何人都不会找到答案。

直树和绘理子大感失望，但智衣仍不放弃。

「你真的不知道吗？可是红华她好像知道。」

听到智衣的话，里佳稍稍皱起眉头。

「既然神乐同学知道，大概是从艾略特神父的笔记里看到的吧。」

「艾略特神父？」

「啊，也对。高村同学你们不知道他的名字呀。就是把村子变成这样的那位传教士，他就叫做艾略特神父。」

就是那位六十多年前来到本村，结果不小心吸了村民血液的头痛吸血鬼。

「如大家所知，感受到责任的艾略特神父已经为了寻找让吸血鬼变回人类的方法而外出旅行，但他也留了笔记下来。那就像是给村民们的提醒。例如上头提到要注意阳光，或是不可以接近危险的大蒜等等。」

里佳一边回忆一边说着。

「也许笔记上有提到那个疾病。如果是这样的话，神乐同学会知道也不奇怪了。毕竟笔记就是由她们家保管的，她应该有机会看过。」

「为什么在红华家？传给大家看不就好了？」

面对智衣理所当然的疑问，里佳稍微开了她玩笑。

「因为笔记是用英文写的呀。很多人就算看了也看不懂吧？利根崎同学。」

明明当了十几年的高二学生，但智衣的英文还是无法提升，她感到无话可说。

「而且笔记传来传去一定会污损的。所以村民们拿到的是翻译好的版本。负责翻译的人也就是红华同学的父亲，想必只有他看过全文吧。」

但，里佳毫无担忧地笑着。

「就算笔记上真的有提到那个疾病，也一定是不严重的病，因为根本就没有翻译出来呢。」

大概是因为里佳并不知道季代实已经在昏睡了，才没有把这件事情当一回事。

此时，里佳像是想起某事，看了手表一眼。

「糟糕，我还要去开教职员会议呢。我先走啰。」

里佳就这样匆忙地离去了。

被留在原地的直树等人面面相觑。

「你们觉得小平老师的看法如何？」

「绘理接受。神乐学姊一定就是看了原始的笔记才知道那个疾病的。」

绘理用力点头。

「如果那段内容没有被翻译出来，一定是因为有某些秘密。所以神乐学姊才会摆出那样的态度。」

直树也抱持相同的意见。季代实目前的状态就连被红华用力拍打也不会醒来，他不认为这是可以坐视不管的病。

「我不认为去问红华她老爸就能得到答案。既然这样，方法就只剩下一个，那就是去得到那个笔记。」

「你说得到那个笔记，东西是在会长家里耶。就算向他们借也不可能借得到……该不会想用偷的吧？」

「笨蛋，怎么可能做那种事！」

尽管智衣用力否定，她的脸上却浮现出，宛如忽然想到了某种无聊恶作剧的孩子王会有的笑容。

「我们就只是偷偷溜进红华家，然后神不知鬼不觉地把东西借出来而已。看完之后再还回去就好啦。」

她的主意跟小偷没两样。

「那样子太糟糕了啦，对不对，绘理子？」

直树原本想徵求绘理子的同意，但糟糕的是，就连绘理子脸上也浮现相同的笑容。

「绘理一直很憧憬美少女怪盗呢，人家好兴奋哟。」

对于生活长久不变的吸血鬼而言，变化一直是他们所渴求的。也因为如此，想扑灭被点起的好奇心火焰几乎是不可能的事情。

为了解救季代实这个正当的理由，正好可以拿来玩怪盗游戏。她们怎么可能不跃跃欲试。

直树想到自己这一年来的生活已经了解了吸血鬼的习性，只能大大地叹口气。

4

村内望族神乐家的位置座落在高地上，那是一间风格独特的西洋住宅。即使年代已久，反而更凸显出其品味。

时间来到晚上八点钟，一群年轻人正屈身躲在宅邸大门旁的小树丛里。

直树等人全身上下都穿着黑灰色的运动服。尤其是绘理子，她甚至连宛如注册商标般的蝴蝶结都改成灰色的。

然后，除了直树、智衣和绘理子之外，居然又多了一名新成员。

「为什么我也要一起来～」

这位表情像是快哭出来的新成员，是个以大眼镜与大胸部为特色的女孩子。

她的名字是泽田绫菜。治水高中的三年级学生，同时也在学生会担任书记。时常待在红华的身边。

「有什么办法？只有你比较清楚红华家里的构造啊。」

「就是呀，只有泽田学姊平常会到神乐学姊家。想要达成这个计画，泽田学姊的力量是不可或缺的。」

两人热情地游说着绫菜。

「而且绫菜你也很担心季代实吧？很在意她的情况吧？」

「这……我的确很担心。没想到她居然会昏睡不醒……」

绫菜显露出忧心的表情，她原本就是如此善良的人。

「虽然不知道红华为什么要这么神秘，不过我们身为朋友，无论如何都要搞清楚季代实得了什么病啊！」

智衣持续进行说服。

就算不能完全接受，也许还是受到打动。绫菜带着困扰的表情点头。

「所以说，绫菜。如果你的消息是真的，红华的老爸今晚会出门对吧？」

「是的。今天是村子里的高层人物们开会的日子，我想伯父应该快出门了。」

「所以会长的父亲是村子的高层人物吗？真了不起。」

听到直树这句话，智衣投以白眼。

「直树，你都已经住在村子里快一年了。到现在还不认识红华的老爸神乐刚三吗？」

突然有个听起来严厉无比的名字登场了。

「我也多少听说过会长的父亲，神乐家原本统治着这个村子对吧？所以，至今会长的父亲在村子的营运上还拥有很大的发言权。」

「根本不是什么发言权的问题，他几乎可以说是这村子的地下村长。」

「虽然伯父没有担任什么特别的职务，但是据说他拥有关于村子营运上的一切决定权。不过，以村长先生为首的高阶成员们本来就是村子里的老成员，会听从伯父的话也是很自然的。」

绫菜补充说明。

「啊，门好像要开了！」

绘理子小声喊道。

神乐家的铁门开启，一辆高级轿车开了出来。

和直树不同，夜晚中依旧看得见东西的智衣等人带着笃定点了点头。

「很好，头目角色坐在车子上。」

「意思是说，家里就只剩神乐学姊一个人了呢。」

「咦，是这样啊？」

面对直树的询问，绫菜加以回答。

「这里只有会长和伯父两个人住。虽然还有其他佣人，但晚上他们都会各自回家。」

了解到红华也与自己一样是个单亲家庭，直树感到一股莫名的亲近感。但现在并没有时间容许他沉浸下去。

「好，我们走吧！」

从草丛里爬出来的智衣，缓缓地溜进大门的缝隙。绘理子与绫菜也跟在后头。

看了点着零星灯火的宅邸一眼，绫菜开了口。

「浴室的灯是亮的，我想会长应该在淋浴吧。」

「那就可以慢慢找笔记啦。」

智衣对这个好时机感到非常高兴。

「不过，我们要怎么进去呢？」

「包在我身上吧。」

智衣靠近一个没有点灯的房间窗户，然后举起右手。

她的眼睛染成鲜红色，牙齿与指甲化为吸血鬼的尖牙利爪。她将尖锐的爪子放到玻璃上，恐后缓缓划了一个圆，响起一道尖锐的声音。

「好，差不多就这样吧。」

最后智衣朝圆心轻轻敲了一下，被划破的玻璃就这样掉入房内。接下来只要把手伸进去开锁就行了。

「看，开了。」

看到智衣得意洋洋地挺胸站在全开的窗户前，直树没好气地说道：

「怎么把吸血鬼的力量拿来作这种事情。」

「有什么关系嘛，就是这种时候才该用。别管那么多了，赶快进去吧。」

智衣从打开的窗户溜进房子内部，直树等人也跟着进去。

「绫菜，如果有笔记的话会放在哪里？」

「应该会在伯父的书房里吧，在二楼最内部的房间。」

大概是认为事情已经无法回头了，绫菜回答得很干脆。

「好，走吧。」

智衣率先离开了房间，绘理子和绫菜则跟在她的身后。

伤脑筋的是直树。因为房间内太暗了，身为普通人类的他根本就看不见。

「你、你们也等等我啊。」

他慌慌张张地想追上三人。虽然过程中他的脚趾头撞到类似家具的东西，不过最后还是勉强离开了房间。

走廊上有西洋式油灯形状的电灯点亮着。尽管有点昏暗，但至少还看得见东西。

已经看不见智衣等人的踪影，看来她们早就跑上二楼了。

直树似乎感到一点罪恶感，他脱下鞋子并且拿在手上，蹑手蹑足地朝二楼前进。爬上宽大的楼梯后，走在二楼走廊上的直树察觉到一个声音。那是淋浴的水声。

（还以为浴室都会在一楼，原来这里不一样啊。真不愧是西洋建筑。）

虽然直树不知不觉中如此感慨着，但他马上想到另一个更惊人的事实。

发出这个声音的位置，就是红华正在淋浴的地方。

她毫无疑问是全裸。

直树忍不住回想起夏天时曾被红华逼着看她穿泳装的模样。尽管胸部大小比不上绫菜，但她的身材依旧傲人无比。包括纤细的腰围在内，她的体型也和本人同样亮丽出众。

直树感觉到自己的心脏急速跳动着。虽然他用力摇摇头，打算继续前进，但是却发生了出乎意料的悲剧。

淋浴声居然停止了。

虽然知道目的地书房是在二楼内部，但他并不知道详细的位置。如果找到一半红华就跑出来的话就惨了。被她撞见这样的场面，谁知道会受到什么惩罚。

如果这里是日式建筑，直树就不会这么慌了。想必对方会花些时间擦干身体并穿衣服才出来。但这里是一栋西洋风格的高贵宅邸，搞不好红华会围着浴袍出现。

（之前我在外国电影里看过，虽然那看起来像衣服，其实只是浴巾而已啊。她们总是围着一件浴袍就若无其事地离开浴室。）

直树不好的预感成真了，他听见转动门把的声响。

（哇！）

直树勉强把叫声吞回喉咙里，然后朝附近一个开着门的房间里冲去。他藉着走廊上的灯光确认了昏暗的房间里有个像是沙发的影子，于是立刻就躲到那个物体的后方。

但他没能松一口气，房间里的电灯很快就被打开。有个人走了进来。

直树理解到自己陷入最糟的情况，这里竟然就是红华的房间。

和他猜的一样，红华正穿着浴袍。她把头发卷在肩膀上，应该是为了在淋浴时不弄湿。

来到房间里一个大化妆台前的红华，缓缓坐到椅子上。她开始在脸上涂起一些乳液。

在相隔不远的沙发后方，直树正拼命地屏住气息。他全身僵硬着，就怕发出任何一点声响。

此时，还有一阵香皂的味道在刺激着直树的鼻子。

直树感觉到自己的心脏又在加速了。看来躲在沙发后面所带来的不道德感，更加让他感到兴奋。

（会、会长她现在只披着一条浴袍，搞不好底下什么也没穿……）

直树感觉到自己的脑袋深处开始发热，下一瞬间，他的鼻子内有黏稠的液体流了出来。

不必多说，当然就是鼻·血♥

吸血鬼可说是人类的天敌、猎食者，当然不可能放过血的味道。

「！？」

红华瞪大眼睛，并且染成鲜红色。她一语不发地起身，然后走到直树躲藏的沙发前。光是右手轻轻一挥，就把厚重的沙发撞飞到房间角落。

当然出现在她眼前的，就是如兔子般颤抖的直树。尽管他拼命地按住自己的鼻子，当然是

于事无补。

「哎呀，还以为会是谁，原来是高村同学。」

红华眯起眼睛。

「居然会在这种时间独自跑来我的房间……难道说你已经下定决心要让我吸血了吗？」

看到全力摇头否定的直树，红华稍微露出疑惑的表情，但很快又浮现笑容。

「既然如此，你就是来袭击我的啰？那么就算被我反扑也不能有意见，对吧。」

红华带着微笑的嘴角边，稍稍露出了白色的尖牙。手指前端也出现了锐利的指甲。她鲜红色的双眼正在闪闪发光着，看起来仿佛是眼前出现了一顿大餐的猛兽。

直树思索着，希望能找到一些理由蒙混过去，但思考却无法跟上。即使变成吸血鬼的红华让他感到恐惧，但更要命的是她的穿着。

红华只穿着一件浴袍，她的胸前略开，乳沟正在吸引着目光。只要再往深处看一点，恐怕就能一窥整个丰满的胸部了。

鼻血的量又增加了，正不断地从直树按住鼻子的手指缝隙中流出。而这样就更刺激了红华身为吸血鬼的本能。

「高村同学，我会马上让你感到舒服的哟。」

面对呼吸急促、像是马上就要扑过来的红华，直树忍不住大叫。

「我是来找笔记的！神父先生留下来的那本笔记！」

5

在神乐家的接待室里，穿着浴袍的红华正环着手臂伫立着。

她的视线下方，是被施以正座处罚、非法侵入的四人。

由于直树全盘招供的缘故，所有人就这样遭到一网打尽。

「绫菜同学，没想到居然连你也有份，真是太教我惊讶了。」

「对、对不起。」

绫菜一副快要哭出来的表情。

「有什么办法？还不都是因为你不告诉我们。」

智衣嘟着嘴，摆出一副反抗的态度。

「我们也很担心季代实啊。一听说摆在你家里的笔记可能有提到病情，我们怎么可能不管！？一切都是因为友情驱使我们前进！」

听到智衣如此主张，红华只是冷眼看待。

「哎呀，真是如此吗？我还以为你们只是把川澄同学的事情当成藉口，实际上只是在玩小偷游戏而已呢。」

就因为几乎都被说中，智衣一时答不上话来。不过她很快又大声反驳。

「你说的多少也是有啦。不过，我们也是真的很担心季代实啊！」

「就是呀，神乐学姊！请你快说！川澄学姊到底是怎么了？吸血鬼的昏睡症又是什么呢？」

绘理子也以认真的眼神询问。

红华大大地叹了口气。

「总而言之，我先去换衣服。一直穿成这副模样也不是办法。」

她离开了接待室。

「总之你们乖乖在这里等，知道了吗？」

只留下这句话。

原本紧张的气氛一下子获得舒缓，智衣马上就把腿打直结束正座。

「真是，都是直树害的啦。要不是因为你被红华抓到，事情就不会变成这样了。」

「还不都是因为你们把我丢在乌漆抹黑的房间里，我什么东西都看不到，根本就没办法啊。」

「就算被抓到，你就不能设法拖一点时间吗？我们好不容易看找到好像藏有笔记的保险箱。」

「只要再慢一秒招供，我现在就变成村子里的一员了啦！」

了解到那就代表直树的血被红华吸走，智衣即使还有不满，也只能停止抱怨。

智衣也非常想吸直树的血。光是听说可能会被红华抢走，就无法责怪直树招供。

「算了，没办法。」

像是下了决心，智衣又盘腿坐到地上。

「既然红华已经讲明要我们在这里等，她八成是想训我们训到早上吧。既然如此我们就听她讲个够吧。」

就在所有人都多少有了心理准备的时候，换完衣服的红华回来了。她身上穿着看起来很高级的睡裙以及长袍。全身上下都充满着成熟的魅力。

坐到沙发上的红华，把手上拿的东西放到桌子上。那是一本看起来很老旧的笔记本。

「这就是艾略特神父的笔记。」

听到这句话，直树等人发出惊叫。

「红、红华，你到底是怎么了？居然会把这东西拿出来！」

「绘理我们真的可以看那个笔记吗！？」

看到智衣和绘理子大惊小怪的反应，红华叹了口气。

「又有什么办法，我很明白你们担心川澄同学的心情。何况，如果未来又像这样被你们摸进家里来，我可受不了。」

红华向直树抛了个媚眼。

「不过呢，如果是高村同学的话，我可是大大欢迎，随时都可以偷跑进来没关系。就像刚才那样袭击我。」

「给我等一下！直树！你真的那么作了！？」

智衣大吼。

「看到鼻血时绘理就觉得可疑！太龌龊了！高村学长是禽兽！是好色鬼！」

绘理子也跟着批判直树。

「毕竟会长穿浴袍的模样太有魅力了，我知道这也是无可奈何的事情。不过，我还是对高村同学有些失望呢。」

就连绫菜也对直树投以轻蔑的眼神。

「等、等等啊！那当然是会长在开玩笑啊！泽田学姊，拜托你不要用那种眼神看我！」

看到直树拼命主张无辜的模样，红华忍不住笑了出来。

「我只是稍微作弄他而已，高村同学才没有那个胆子。」

虽然很高兴误会解开了，但听到这种话没有男人会觉得高兴。即使她说直树没那个胆子是事实。

「好了，你们想知道川澄同学的病因吧？就在这里。」

红华拿起笔记本，并且开始翻页。她把其中两页摊给直树等人看。

四个人就这样一同凑到笔记本前，注视着上头的文字。

头一个举白旗投降的人是智衣。

「呜呜呜，根本看不懂上面写些什么。」

笔记本上记载着密密麻麻的英文，而且还是难以阅读的草写。

「绘理也是……我的英文不太……」

绘理子也跟着举白旗。

「直树，你看得懂吗？」

「不用问也知道吧？从我的英文成绩来看。」

直树摇摇头，像是在举手投降。

就只有一个人，绫菜正以认真的表情注视着笔记。像是在循着文字列阅读，眼镜底下的双眼正在左右移动着。

「明明都偷跑进来拿笔记本，却没有半个人看得懂，这样实在太蠢了。绫菜，拜托你了。」

智衣擅自开始期待绫菜。

「怎么会……这么一来季代实同学她……」

读完文章的绫菜几乎发不出声音。

「喂，到底是怎么回事？绫菜，快点告诉我们啊？」

「泽田学姊！请问川澄学姊到底得了什么疾病！？」

智衣和绘理子质问绫菜，直树也以严肃的表情注视着她。

「吸血鬼的昏睡症。那是一种吸血鬼的罕见疾病。原因和预防方法都不明。一旦得了这种病，吸血鬼就会陷入沉睡。如果没有获得治疗，将会永远无法醒过来。」

「永远无法醒过来！」

智衣表情僵硬，但身旁的绘理子还抱着希望。

「可是可是可是，既然说如果没有获得治疗，就表示有治疗法对不对？到底该怎么作才好呢？难道很难治疗吗？」

「治疗法并不困难，只需要，一个吻就好了。」

由于突然冒出『吻』这个无法预期的字眼，智衣、绘理子及直树都没能意会过来。

「异性吸血鬼的吻，那就是治疗吸血鬼昏睡症的特效药。只要在发病后的一星期内作到就可以了。」

尽管听起来是个不合常理的治疗法，但吸血鬼本身就已经是不合常理的存在，就算出现这种像是童话故事般的治疗方法也不奇怪。

「意思是说，川澄只要在一星期以内让其他男性吸血鬼亲吻，就会醒来是吗？」

直树的脑袋里浮现了班上男同学亲吻昏睡的季代实的景象。突然间，他感觉到内心不太舒服。

（嗯，怎么了？明明找到治疗法应该很高兴才对。为什么我会觉得不开心呢？）

「这对川澄学姊实在很过分呢！」

绘理子像是在同情着季代实。

「可是，如果没有其他方法的话也只能这样了。就只能要季代实当作被野狗咬了，反正就只是一种治疗而已，一了百了。」

尽管智衣轻笑着，但绫菜却带着哀伤继续说下去。

「问题就在于，不能一了百了。」

「咦？」

所有人的目光都集中到绫菜身上。她则带着感到困扰的表情看向红华。似乎像是在询问她能不能说出来。

看到红华无言地点头后，绫菜才开口。

「从吸血鬼的昏睡症中醒过来的人，将会彻底迷上让自己醒过来的人。而且她的这份爱意将永远不会消失。」

「你、你说什么！？」

直树不禁大叫。

「这么说，川澄想要醒过来，就一定要迷上某个人才行吗？」

「是，事情就是这样。」

绫菜深深点头。

「据说很久以前，吸血鬼拥有控制吸血对象的能力。但笔记本有提到，这种疾病大概就是受了那个已退化的能力所影响。」

红华无力地摇摇头。

「所以，你们知道我为什么要隐瞒川澄同学的病情了吧？如果这个疾病和治疗法广为人知，一定会有一大群笨男生涌向川澄同学。所幸这个疾病只有极少数的村民知道，我们绝对不可以泄漏这个秘密。知道了吗？」

「可是，我说红华。时间限制就只有一个星期耶？该怎么办？」

面对智衣的询问，红华答得十分苦涩。

「还能怎么办，就只能在那之前决定人选而已。选择去亲吻川澄同学，接受她永恒爱意的人选。」

红华带着叹息作出结论。

「到底要选谁，真是一个很困难的问题。为了川澄同学着想，一定要选个好人选才行。」

6

直树、智衣、绘理子及绫菜四个人，一同行走在治水村昏暗的夜路上。

由于他们离开红华家已经是晚上十点以后的事情，现在已经接近十一点钟了。

四个人都没有说话，就只有沙沙的脚步声在黑暗中响着。

「是说，季代实那家伙还真是得了麻烦的病啊。」

或许是无法忍受沉默，智衣先拨动了夜晚沉静的空气。

「就是啊。必须靠亲吻才能醒来，而且还会迷上亲吻的对象，简直像是睡美人一样。」

直树嘟嚷道。

「只剩下一个星期而已呢，到底是谁会被选为川澄学姊的白马王子呢？」

「既然会长已经说要严加挑选了，应该会是个可靠的人选吧……」

「可靠的男生……吗？至少能确定不会是我们班上的那群人吧。」

智衣不屑地说道。

「不如由利根崎学姊去亲吻吧？你这么像男人，搞不好会有效哟。」

「这我也想过了。可是就算季代实真的醒过来，我还必须接受她的爱才行。」

就算只是绘理子的玩笑话，智衣也严肃地回答。

「那就是问题呢～就算再怎么可靠，也不可以挑选有太太或女朋友的人。如果不是能和季代实同学成为相爱的情侣，对彼此都会是悲剧。」

「意思是说，现在单身，和季代实成为情侣也没问题的男人吗？」

「学校里是有很多。毕竟川澄学姊长得很漂亮，其实有很多男生喜欢她。」

「问题就在，全都是一群笨蛋吗。」

智衣叹了超大的一口气，直树突然提出一个问题。

「欸，关于刚刚的话，川澄很受欢迎吗？」

「什么啊直树，难道你没发现吗？」

智衣受不了地看着直树的脸。

「那家伙还满受欢迎的啊，有很多男生很迷她。」

「可是，川澄平常不是很冷淡吗？」

「高村学长，这你就不懂了。就是有很多男生喜欢那样冰冷的女孩子呦。」

绘理子故作成熟地说明着。

「季代实同学刚成为村子里的一员时真的很辛苦。听说她的鞋柜总是有情书，放学后想向她表白的男生甚至得排队呢。」

「而且，她全部都斩钉截铁地拒绝了，现在所有人都已经知道自己没有希望，才会这么安分。我们班上那些笨男生也有好几个人尝试过了。尤其是熊谷之类的。」

智衣把其中最具代表性的笨男生拿出来举例。

「就因为这样，要是他们听说季代实的病，一定会造成大混乱。红华说的没有错，肯定会有很多臭男生想来抢夺季代实的嘴唇。所以一定要保密才行，知道了吗？」

所有人都点头同意智衣的话。

「啊，绘理的家就在前面，绘理先告辞了。各位晚安。」

绘理子朝大家低头行了个礼，然后就轻快地消失在黑暗中。

「我也要往这条路了，各位同学晚安了。」

接着绫菜也离开了。

再走一段路后碰到的岔路上，智衣停下脚步。

「怎么办？直树。如果怕太黑的话，我可以送你回家喔？」

「不，没关系。有月光照路，回家不会有问题。掰——。」

当直树挥挥手并转身踏上归途时，智衣在后头朝他喊了一声。

「直树，如果我也得了和季代实一样的病，那个时候……」

「咦？」

看到直树回头，智衣又急忙挥舞双手。

「什么也没有啦，你要早点睡喔！」

智衣几乎是用喊叫声回答，然后她就朝自家位在的村公所大道区跑去。

「怎么搞的？」

虽然这让直树感到疑惑，但他还是踏上回家的路。

这座村子里的路灯非常少。虽然在夜晚行走非常不方便，但幸好今晚天空没有云朵，有月光可以照路。

而且，他已经开始习惯像这样走夜路了，直树顺利地回到自家。

「我回来了。」

打开位在郊外的独栋建筑大门，直树小声地说道。

「喔，你回来啦。」

一名穿着皱巴巴工作服的消瘦中年男子从客厅里探头，他就是直树的父亲高村直太郎，职业是无名科幻小说家。明明早就超过截稿日了，他还像这样悠哉地喝着酒，满脸通红。

不过，直树现在并没有力气去质问他。

「老爸很高兴喔，你终于也学会夜游啦。」

直树毫不理会在说些奇怪话的父亲。他自顾自的踩着吱吱作响的楼梯爬上二楼，前往自己的房间。然后就在宽阔的榻榻米房间角落的铁床铺上躺下。

他现在满脑子都在想着季代实的病。到底会是谁被选上，这让直树非常在意。

（虽然不太可能，但我们班上的男生会被选上吗？）

他想起那些看着清凉杂志并发出怪声的男同学们。

（要是迷上那些家伙可就糟了，一定会被要求一些奇奇怪怪的事情。）

被迫穿上学校泳装的季代实。

戴上猫耳，招手摆出『喵喵』姿势的季代实。

被要求穿着裤袜并踩对方脸的季代实。

被命令换上女仆装，还得故意打翻红茶的季代实。

直树如此幻想着。

而且他的幻想还变本加厉，甚至开始想着一些不能说给未成年小朋友听的内容。

红着脸的直树喊叫着。

「如、如果会变成那样的话，还不如让我去亲她，至少我就不会要求她做那些事情。」

如果要治疗她的病，必须由异性吸血鬼来亲吻。现在虽然直树还是人类，但他想成为吸血鬼也很容易。

（只要拜托智衣、绘理子、泽田学姊或会长，要她们轻轻咬我一口……）

但是，直树又用力摇头。

（我到底在想什么啊，一点都没考虑到川澄的心情。）

虽然他自认为和季代实的感情还算良好，但那并不代表季代实对自己拥有特别的感情。

尽管对方已经那么常和自己在一起，也非常照顾自己，但直树就是不那么想。

原因就在于刚来这座村子时所发生的事情。他以为自己受女生欢迎而得意忘形，最后落得被追着吸血的凄惨下场，这已成了他的心灵创伤。

（虽然应该没有被讨厌，但她那么照顾我还是为了吸我的血吧。这种情况下我想亲吻她，根本是太不要脸了。完全没有考虑到川澄的心情嘛。）

只要吻下去，让她迷上自己，那么原本的心情一点也不重要。即使这样的想法闪过脑袋，但直树还是用力摇头。

季代实真的很照顾直树。所以直树既感谢也信赖着她。平常冷静的她经常解救直树面临的危机，而且不只是直树，就连从东京前来的朋友们也被她救过。

如果不是季代实灵机一动，不只是直树，恐怕连来游玩的朋友们也都要被吸血了。

因此，他无论如何都不想怱视季代实的心情。

「如果亲吻的人，刚好就是川澄喜欢的对象就没问题了说。」

直树不经意地说了这句话，然后整个人从床铺上跳起来。

他瞪大双眼，然后用力大叫。

「就是啊！这不就是方法吗！只要找到川澄喜欢的对象，让他亲吻川澄就好啦！如此一来就不需要迷上自己不喜欢的对象了，不是吗？」

自己虽这么说，但这是非常困难的任务。以季代实的个性而言，很难想像她会随便把自己喜欢的对象告诉别人。

就连乎常很常在一起的直树都没有听说过，甚至连一点点的迹象都看不出来。

如今她本人陷入沉睡，在不能亲口问到的状况下真的有办法找到人吗？

尽管抱着这样的不安，但直树还是下定了决心。

「为了川澄着想，一定要找到人。」

直树紧握拳头，坚定地说道。

7

然后，时间来到隔天放学后。

直树来到戏剧社的社办，事先约好见面的数名社员已经集合在此了，全部都是女生。

「到底是怎么了？高村同学。为什么要我们在社办集合呢？」

其中一名社员问道。

「难道说，你终于决定要入社了吗？」

「啊，那真是太好了，社长也一定会很高兴的。」

由于直树经常受季代实之托前来帮忙修理一些大布景，社员们都已经认识他了。

尽管社员们欢欣鼓舞，但其中一个人还是不忘询问直树。

「对了，社长今天是不是请假？我有一些预算的问题想找她商量，可是她好像不在教室。高村同学，你知道是什么原因吗？」

「呃，记得她好像有说过要帮家里的忙？会请假几天之类的。」

直树把头转向一旁，装傻地答道。再怎么样也不能告诉她们季代实正昏睡着。

在被追问而拆穿之前，直树抢先开口。

「之所以会找大家来，其实是有事情想问。」

「有事情？」

「是啊。因为我觉得川澄在这里应该比在教室里更常说话，所以才会来请教你们。」

先作了这样的开场白后，直树以极为严肃的表情问道。

「各位知不知道川澄喜欢的人是谁？」

社员们的表情先是一愣，一下子又转变为戏谑的笑容。

「哈哈哈，原来是这么一回事呀！」

「直树想和社长表白是吗，所以才这么在意。」

「可是，我认为不应该在意那种事情，要拿出『不管她喜欢谁都无所谓！要让她迷上我』的气魄才行。」

「哇，孝子好帅～～～～～」

看到社员们愈闹愈开心，直树连忙挥手否定。

「不，你们误会了，事情不是那样的。」

「不然是怎么样？」

但这问题又让直树无法回答。

「唔唔……」

看到直树答不出来，社员们纷纷朝他点头。

「没关系的，我们不会告诉任何人。」

「对呀对呀，高村的心情我们对社长也会保密的。」

到底要如何才能在不说出真相的前提下，解开误会呢？直树想了一会儿，但没有找到答案。

（没办法，总之先当作这么一回事吧。）

下定决心后，直树向众人低头恳求。

「拜托！请告诉我！因为我真的很在意！」

面对直树真诚的态度，社员们互望了一眼。

「欵，有人知道社长喜欢谁吗？」

「我不知道耶，因为她从来都没有提过。」

「我也是。之前我曾经问过社长，可是她一点兴趣也没有，话题一下子就被结束掉了。

「毕竟她平常很少谈私人的话题。」

社员们纷纷环起手臂思考着。

「不过，硬要说起来应该是那个人吧？」

「就是呀，怎么想都应该是那个人。如果不是他，应该就没有任何特别喜欢的人了。」

社员们互相点头。

「拜托快告诉我，你们说的那个人是谁？」

看到直树以激动的神情探身问着，社员们脸上都挂着窃笑。

「要给个提示的话，大概就是她平常很照顾的人吧。」

「就算说那个人是社长身边最亲近的男生也不为过吧。」

「这是我听说的，社长好像每个月都会去他家玩呢？」

「给了这么多提示，应该已经猜到了吧？你说那人是谁呀？」

她们宛如猜谜节目的主持人，向直树询问最终答案。

「川澄平常最照顾，离她亲近的男生，而且每个月还去他家玩……」

直树先是一阵思索，然后又问：

「到底是谁啊？」

下一秒钟，社员们的表情全变了，全都摆出一副受不了的表情。

「虽然早就觉得很迟钝，没想到居然这么木头。」

「我们不想告诉你了，请自己一个人慢慢烦恼吧。好了，我们还有事情要讨论。」

结果直树一下子就被赶出社办。

「闲、请等一下，拜托你们告诉我啊！现在事情很紧急！」

无论直树再怎么大喊，关上的门背后都没有回应。而且她们似乎还上了锁。

「唔～～～！她们到底在说谁啊？」

直树又重新思考一逼。

（川澄平常最照顾，跟她最亲近的男生，而且每个月还去他家玩……）

即使把她们的话再重复一次，直树还是找不到答案。

「明明我很常和川澄在一起，怎么会完全没有察觉到呢？这么迟钝连我自己也受不了。」

就在直树深深感叹的时候——

「我问一下，你是戏剧社的人对吧？」

突然有人这么叫他。有三名女学生正瞪着直树。

站在最前面，看起来比较强势的女学生率先问道。

「叫你们社长川澄同学出来！」

「川澄她今天请假，而且我不是戏剧社的……」

女学生没有理会直树的话，自顾自地激昂了起来。

「请假！？我们还特地来找她。」

女学生大怒。

「不然你转告她吧！」

她还是没有在听直树的回答，连珠炮似地把话说完。

「如果想和相场学长亲近，就要好好参加粉丝团。下次不可以再擅自跑来加油了！」

当女学生说完话转身准备离开时，直树全力喊住她。

「给我等一下——！」

「干嘛？」

女学生一脸不悦地回头。

「能不能把事情说详细一点？相场学长是谁？川澄喜欢那个人吗？」

「足球社的社长兼王牌攻击手，你居然不认识相场和也学长，真是太离谱了。他又帅又英俊喔！」

女学生一脸陶醉。

「上星期他们和青年团有一场足球比赛，当然我们粉丝团也都有去加油。川澄同学也在那里，而且比赛结束后她还厚着脸皮地和相场学长说话。讨厌～～～～我绝对不原谅她。」

她激动得要狂踩地面。

「总之你要把我刚才说的话转告她，听到了吗！」

撂下这句话后，女学生们就这样离去了。

「真没想到会得到这么重要的消息，川澄居然有去看足球比赛。」

由于从没听说季代实喜欢看足球，应该不是单纯去看比赛。既然如此……

「也许川澄喜欢的对象，就是那个王牌攻击手相场学长。」

直树眯起眼睛，轻轻点头。

「我就稍微去看看他是怎么样的人吧。」

8

棒球社、足球社和田径社正在校园里从事着放学后的社团活动。即使如此仍不会感到拥挤，真不愧是拥有宽广校园的乡下学校。

直树前往足球社。社员们正分成两队进行练习比赛。

其中一名男学生盘着脚底下的球并且作出强力的射门。那是一记距离相当远的射门，但还是漂亮地进了球。

「呀——相场学长好帅喔！」

女学生们的尖叫声传入直树耳里。仔细一看，那些人正是刚才在戏剧社社办前瞪直树的女学生们。

（刚才射门的人就是相场吗？）

直树仔细观察那个人物。

他应该比直树大一届，身材高大体型也很均衡。原来如此，长相也的确很帅，难怪女生们会为他骚动，这位小帅哥简直像是电视偶像。

相场举起右手，朝着为自己加油的女生们投以微笑。如果是动画的话，他的洁白牙齿大概会因为阳光反射而发亮吧。就是如此阳光的笑容。

「相场和也……吗？可恶，还真的很帅。」

就在直树不太高兴地自言自语之后，守门员踢出的球在空中画了一道曲线，滚到直树脚边。

「抱歉，可以把球踢给我们吗？」

相场朝直树大大挥手。

距离约二十公尺，只要轻轻一踢应该就能踢到。但是，直树却在无意识下把力量集中在右脚。

「嘿呀！」

稍稍浮起的球超相场飞去。相场用脚把球停下来后，吹了一声口哨。

然后，他就抱着球来到直树面前。

「记得你是高村直树吧？」

「咦！？」

看到直树因为被知道名字而感到讶异，相场轻轻一笑。

「因为你是去年夕月祭的目标啊，记得名字是当然的。」

村子里一年一度在七月举办的夕月祭。但真相却是被引来村子里的可怜祭品，被村民追杀的吸血庆典。直树表面上也被视为在那场庆典中被吸了血。

「之前就听说你脚程不错，但控球的技术也不错。你以前练过足球吗？」

「不算是有练过，偶尔帮忙出赛而已。」

「那好像会很有趣。」

相场微笑着。

「如果不嫌弃的话，可以请你作个盘球射门吗？」

「我吗？」

「对，往那边的球门。」

相场指着刚才自己进过球的球门。

「我会要他们作一点守备，应该无所谓吧？啊，当然不会用到吸血鬼的力量，毕竟是村子里的规矩。」

面对相场的询问，直树静静答道。

「我是无所谓。」

「好，那就决定了。」

相场告诉其他社员后，就把抱在怀里的球滚到直树脚边。

「OK的话就开始吧。」

直树默默地点头，然后就踢了球开始奔跑。

他一面盘球一面朝球门迈进。虽然路上有几名足球社社员打算抄球，但他仍以华丽的技巧一一穿越。

当两人同时朝他铲球的时候，他甚至能轻轻以脚尖挑起球，以跳跃的方式过人。

虽然直树平常碰到危机总是被季代实她们解救，给人不是很可靠的印象，但那只是因为他比不过吸血鬼的力量而已。

以人类来说，他的运动神经其实很不错。在某些体育项目上，他能够表现出进入正规出赛成员的实力。

直树突破所有防守后，就以一记不输给相场的长射进球。

「好，差不多这样吧。」

他有点得意的呼了口气，但马上又回过神来。

「不、不对，我干嘛这么热衷。」

理由很单纯，他只是莫名其妙不想输给相场而已。

（就算对方可能是川澄喜欢的人，我也不需要嫉妒吧。）

在直树深深反省的时候，相场跑了过来。

「果然和我想的一样，你真的不错，很有一套啊。」

他把手用力按在直树肩膀上。

「高村，你要不要加入足球社？」

「咦？」

「既然有那么好的技术，不参加社团就太可惜啦。」

相场热情地劝说着。

「我、我对足球又没有兴趣。」

直树像是不屑一顾，还把头转到一旁。

「是喔，那还真可惜。不过你可以再考虑看看，反正时间多得是，如永远那么长。我会期待你改变心意的。」

相场拍了拍直树的肩膀，然后就朝社员们喊道：

「好，稍微休息一下吧。」

相场和社员们一起移动到校园旁的长椅上休息。

被留在原地的直树，注视着相场的背影。

即使直树刚才的态度很失礼，但相场却一点厌恶的表情也没有。就算是身为男人的直树来看，他也是个不错的人。

长相英俊、会踢足球、有领导能力，又很有朝气，就算季代实喜欢他也不奇怪。

但是，直树的内心就是无法接受这个事实。就因为对方太过完美，无论如何就是不觉得舒服。

「可、可能性的确是很高，但也不能就这样断定。一定要有不可动摇的铁证才行。」

尽管嘴巴上这么说，但直树完全无法想像什么叫做铁证。

（唔～～真的会有东西能够证明川澄喜欢那个叫相场的人吗？）

当直树环起手臂思索时，一道声音传了过来。

「太、太厉害了！高村同学！」

缓缓跑过来的人是绫菜。她来到直树面前，以兴奋的语气说道。

「原来高村同学这么会踢足球呢！我刚才看了好感动！真的、真的很帅气呢！」

绫菜看向田径社。

「利根崎同学好像不再，真遗憾。如果她看到一定会很高兴的。」

尽管绫菜像是有些遗憾地皱着眉……

「不对，也许这样比较好。毕竟只有我看见高村同学帅气的模样。啊，没事没事！」

她一会儿高兴，一会儿消沉，一会儿又不好意思，看起来很忙碌。

「啊，我忘记正事了。会长托我来找高村同学，刚才班上的同学告诉我说看到高村同学在校园里。」

绫菜先确认了四周有没有人会听到后，才小声地告诉直树。

「会长已经选了几个亲吻季代实同学的人选，想找身为男性的高村同学询问意见。」

直树没有听清楚，因为他还在思索着铁证的事情。

「泽田学姊！」

他突然大声说话，然后注视着绫菜。

「请问你有喜欢的人吗！？」

「咦！？」

绫菜大吃一惊。

「怎、怎么突然问我这个，真是教人惊讶呢！」

绫菜先是红着脸，姿态忸忸怩怩，然后才点头。

「是，我有喜欢的人。」

尽管她偷瞄了直树几眼，但直树当然不可能明白她的视线有何含意。

「那你有证据吗？」

「证据？」

「我知道这样问很奇怪，可是我想当作一种参考。就算以毫无关联的第三者立场，也能知道泽田学姊喜欢那个人的证据，请问会有这种东西吗？」

即使面对这种不清不楚的问题，依旧会认真思考，这就是绫菜的优点。她把食指放到嘴唇思考。

「证据、证据、证据……啊！」

绫菜大叫。

「有证据，就是日记。」

「日记！？」

「是的，其实我每天都有写日记。虽然没有把关于喜欢的人的事情告诉别人，但有写在日记上。所以日记应该能够当作证据。」

的确那可以说是决定性的证据。虽然前提是季代实有在写日记，而且上头还写了她喜欢的对象。

「顺带一问，每个女孩子都会写日记吗？」

「很难说呢，我的朋友当中也只有几个人。但就算不写日记，偷偷收藏爱慕男性相片的女孩子应该不少。」

看来很有一查的价值。

因此，目的地就只有一个，当然就是季代实的住家。

「高村同学，你没有说想要看我的日记呢。那个，如果高村同学真的这么想知道的话，我没有关系的。毕竟我也没有勇气说出口，还不如让你看……」

尽管绫菜满脸通红，又开始露出害羞的态度，但可怜的是直树完全没在听。

「对不起，泽田学姊，我还有事情先告辞了。谢谢你的帮忙！」

直树只丢下这句话，就朝校门的方向跑去。

9

离开学校后，直树在乡间小路上轻快地跑着，二十多分钟后，他留在一间房子的门口。虽然格局不大，但房子看起来很新。设计很现代化，看起来就像都会住宅区里会有的住宅。

在看起来时髦的门牌上，有着『KAWASUMI（川澄）』的英文。没错，这里就是季代实的住宅。

「我到了。」

直树稍稍缓和了一下因为全力奔跑而变得急促的呼吸，然后重新整理作战计画。

（为了找到不可动摇的铁证，首先必须踏入川澄的房间才行。只要以会长托我前来为由，应该就能进入她的房间吧。）

直树按下门铃。虽然他知道住址，但这还是第一次前来拜访。

不到一会，门后传来一名女性的声音。

『是，请问是哪位？』

「啊，不好意思，我是川澄……不对，我是和季代实同学同班的高村。」

『咦，你就是和她同班的高村直树同学？』

对方的声音带着惊讶。

「是的，就是我。」

几秒钟内，大门就开了。

现身的是一位看起来很和蔼的年老女性，她的身后还有一名男性。他们应该就是季代实的养父母。季代实是这对夫妇所收养的养女。然后，她在前年的夕月祭中被吸血，成为一名吸血鬼。

可是，季代实并不是单纯被当成极品而叫来这座村子。这对夫妇真的很想要孩子，才会选了季代实。听说这栋房屋也是特地为了季代实而盖的。

「初、初次见面，我是高村直树。」

这是他第一次和季代实的养父母见面。直树抬头挺胸，恭敬地向两人打招呼。

但直树的手却突然被男子拉住，而且还把他拉进房子里。他被带往接待室，面前还摆了一杯牛奶。

「哎呀，你就是高村直树同学啊，我早就想看看你了。」

男子满脸笑容。

「就是呀，我们有告诉季代实，要她带你过来家里一趟。」

女子也以笑容接待。

「喔……」

看到直树因为突然受到热情款待而发愣，女子温柔地向他说明：

「季代实常常提到高村同学。她说你常常帮忙戏剧社的工作。而且她也常常去高村同学家打扰，真的很谢谢你。」

「不会，别这么说。」

每当直树的身分快要穿帮时，都是季代实帮忙掩饰，现在甚至还帮直树剪头发。对于一直受她照顾的直树而言，这样的道谢实在让他承担不起。

「真的，她很常提到你啊。」

男子也跟着开口。

「她是很文静的女孩不是吗？我们相处了一年左右她的话还是很少。虽然知道她不是讨厌我们，但还是令人挺寂寞的。不过，自从你在去年夕月祭成为村里的居民以后，她就常常在家里谈到你的事情。我想她应该是因为吸了你的血，所以很在意你的事情吧。」

直树回想起有这么一个套好的设定。

「可是，没想到她却得了这种病。」

女子的声音变得悲伤。

「啊啊，就是啊。当神乐家的大小姐把病情告诉我们的时候，眼前简直是一片漆黑啊。虽然大小姐说过会负责替她找个好对象，但我还是很不放心。」

男子无力地摇头，但是不知为何又以某种松了口气的眼神看着直树。

「不过还好现在有你来了。」

「真的。」

女子甚至感动得泪眼汪汪。

「老实说，我已经和妻子谈过了，希望被选上的人是你。」

「毕竟季代实一直很在意你，如果对象是你的话，对她而言应该是件好事。」

「请稍等一下。请、请问你们到底在说什么啊？」

看到直树慌张的模样，两人十分疑惑。

「什么意思，你不是来亲吻季代实的吗？」

「不、不是的！不是那样的！」

直树全力予以否定。

「我只是受会长之托，来看看川澄的情况如何而已！」

听到直树的话，两人都露出失望的表情。

「什么啊，原来是这样。」

「还以为你被选上了呢。」

先是深深叹了口气后，两人又站了起来。

「季代实在二楼房间睡，我们带你过去。」

直树被带往一个位于二楼南边的房间。或许是因为整理得很整齐，看起来十分宽敞。和自己或智衣杂乱无章的房间很不一样。

房间里的床铺上，季代实正静静地沉睡着。就和保健室里看见的没两样。没有呼吸的她，像是时间停下一般地昏睡着。

「是不是都没有变？她一直都这样。」

「原本在想她会不会突然醒来，但怎么摇她或喊话都不会醒来。」

女子先是悲伤地摇摇头，突然又以突兀的语气说话。

「哎呀，亲爱的，后院的草还没拔呢。应该趁天黑之前赶快去拔。」

「这种时候你在说什么啊？就算天黑我们也……」

女子注视着碎碎念的男子，她的眼神像是在诉说着某事。

「啊。」

男子像是意会过来，发出同样突兀的声音。

「喔，差点忘了，必须赶快去拔后院的草。好，我们一起去吧。」

「高村同学，那么我们先失陪一会儿了。你就在房间里慢慢来吧。」

「是啊是啊，你可以慢慢看季代实的睡脸。」

「就算神乐家的大小姐没有选你，也不必客气哟。」

「是啊，我们不会介意的。」

「亲爱的，走吧。」

「说的也是，就让年轻人自己相处吧。」

说着像是在相亲时常听到的话，两人就这样离开了房间。

「他们该不会是要我去亲季代实吧？」

虽然以父母而言有点怪，但被接受并不是件坏事。

直树望着季代实的睡脸。如同在保健室看到的，他真的觉得很可爱。

由于几乎快要产生了不好的念头，直树用力稳住脚步忍耐着。

「我到底是来干嘛的，这样和色狼有什么两样？再说，就算我亲她也没有效果啊。」

他用力拍打自己的脸颊。

「好了，快点完成目的吧。」

直树走向房间的书桌。

「川澄，抱歉了。一切都是为了你好。」

先如此辩解着，直树抱着内疚的心情下定决心，就把手放到书桌抽屉上并拉开。

抽屉里排列着笔记本及备用的文具用品，整齐程度不是直树可以比拟的。

直树翻找着抽屉，专心地寻找证据。但里头并没有发现像是日记的东西。

虽然他花了很多时间翻找所有抽屉，就是没有找到证据。

「看来这一招不管用。」

尽管直树遗憾地叹气，但他也察觉到自己像是松了口气。

「不、不是的！我才不是因为没找到决定性证据才松了口气。」

就在他一边找藉口一边推回抽屉的时候，他的目光停留在与书桌一体成型的直立式书架。

在教科书及字典当中，有一本笔记本混在里头。颜色看起来和一般读书用的笔记本不太一样。

直树打开笔记本，发现那不是用来读书的东西。

上头写的是人名和台词，而且还有表演方式的详细注记。

「原来川澄会自己写剧本啊，真了不起。」

在直树佩服地翻着笔记本时，里头夹的一张照片掉到桌子上。看到照片中的人物，直树瞪大了眼睛。

他的脑袋里突然出现了绫菜的脸，正探头带着微笑进行说明。

『就算不写日记，偷偷收藏爱慕男性相片的女孩子应该不少。』

「终于，找到证据。」

原本该是预期的事情，但直树却感觉到像是后脑被铁锤打中一样。全身无力的直树，坐到书桌前的椅子上。

「果然，川澄喜欢的人就是他。」

直树手上的相片，上头就是刚才与他见面的人物，相场和也。

照片上的他穿着球衣，像是因为进球而感到高兴。

「已经找到决定性的证据了，我不需要再迷惘。为了川澄着想，我只能这么做了吧。」

直树像是在对自己说话，然后又转头看向沉睡的季代实。

「等着吧，我会去说服你心目中的白马王子。这就是我第一次，也是最后一次还你恩情。」

10

隔天午休时间，天空中依旧是乌云密布。

当相场正在三年D班教室里喝着牛奶时，班上同学喊了他一声：

「喂——相场，有人在走廊上等你。就是那个去年来村子里的新人，他叫什么名字啊？」

相场没有理会正在思索的同学，急忙跑出教室。

看到正在等待自己的学弟高村直树，相场露出清爽的笑容。

「高村，你居然会特地来找我，难道你对足球社感兴趣了吗？」

直树以异常严肃的表情注视着相场。

「我有事情想拜托相场学长。」

「拜托我？」

「是，那是一件非常重要且严重的事情。不能在这里谈，请跟我到比较没人的地方。」

「稍等一下。」

相场喊住正打算带路的直树。

「拜托我……啊。」

相场把手放到下巴思考了一番，然后眯眼点点头。

「知道了。那就陪我玩个游戏吧，就当作饭后运动。」

听到相场出乎意料的话语，直树不禁目瞪口呆。

然后……十分钟后。

相场和直树来到校园。

「彼此争球，并且朝对方球门进攻。最后只要先射进对方球门三球的人就算胜利。」

相场以脚拨弄着从体育用品仓库拿来的足球，向直树说明着。

「我想应该不必说你也明白，不能使用吸血鬼的力量。规则差不多就这样吧。有什么问题吗？」

直树不放心地问道：

「只要我赢了，学长就愿意听我的请求吗？」

「虽然还不知道内容是什么，不过我保证答应。可是，如果你输了就要加入足球社。没问题吗？」

「……我明白了。」

「很好。」

相场把球放置在球场中央，然后两个人都向后退了几步。

「等我喊3、2、1、GO就开始，OK？」

和直树作了确认后，相场开始倒数。

「3、2、1、GO！」

两人同时朝球冲去。先抢到球的人是直树。他穿越相场，并且朝球门跑去。

但相场也没有让直树专美于前。他很快就追上直树，并且抢走球。摆脱意图把球抢回去的直树后，他在球门前施展一记强力的射门。球就这样飞进毫无防备的球门内。

「先驰得点啰。」

相场得意地笑着。

「高村，这次让你开球吧。」

直树拿着球回到中央地点，快速地开了球。

相场追上一边盘球一边朝球门进攻的直树，并且积极抢球，但是不像刚才那样顺利。

直树先停球让相场重心朝前，再趁机超越他。然后在被追上前完成一记射门。

「一比一吗，看来好像愈来愈有趣了。」

相场的表情有些僵硬。

有许多学生在午休时间没事可作。尤其是这间治水高中的学生，他们一向过着永无止尽的高中生活。

在校园里突然开始的这场较量，逐渐吸引了前来围观的群众，不知不觉中就聚集了不少观众。

「欸，他们在干嘛？」

偶然路过的学生问道。

「足球社的相场同学正在和二年级的高村较量，好像是先得三分的人获胜。」

「哈哈，高村真笨，怎么可能赢得了。」

「不，现在比赛意外的精彩。原来那家伙的运动神经这么好。」

于是连路过的学生也加入，围观人数变得越来越多。

地点转换，这里是治水高中的学生会室。

红华正和绫菜面对面，努力制作着人选名单。

简单地说，这份名单就是『白马王子候补名单』。她们正在思考要让谁去亲吻季代实。

「总而言之，高中里的人选大致上就是这些。接下来还必须制作二十岁左右，个性比较正经的人选名单。」

「会长，不和老师或村公所的人商量吗？」

面对绫菜的询问，红华缓缓摇头。

「我和父亲讨论过了，他要我不可以告诉太多人。毕竟消息可能从任何地方走漏。现在只有村长先生和校长先生知道，他们都接受由我负责处理。」

红华以严肃的表情继续说着。

「正因如此，我绝对不可以出错。」

就在此时，窗外传来欢呼的声音。

「怎么一回事！？在别人认真想事情的时候吵什么？」

红华一脸不悦地看向窗户。

「有好多学生聚集在校园里呢。」

绫菜起身走到窗边，隔着窗户观察校园的情况。

「好像是在踢足球。」

「意思是足球社在进行练习比赛吗？」

「不是的，他们还穿着制服，看起来不像一般的比赛。而且只有两个人在争球……」

说到这里，绫菜惊叫了一声。

「是、是高村同学！那是高村同学！」

绫菜打开窗户，探头出去看。

「对手是D班的相场同学！」

「和足球社的相场同学比赛？高村同学怎么可能赢得了。」

「不会！没有那种事！」

面对红华的一笑置之，绫菜认真地予以否定。

「高村同学很会踢足球！昨天他也一点也没有输给足球社的社员们！」

或许是起了兴趣，红华缓缓起身来到窗边，并且和绫菜一样探头俯瞰校园。

直树和相场两个人，正在展开一场激烈的攻防战，球就在两人之间来回移动着。

「高村同学还挺有两下子的。」

红华有点惊讶地感叹着。

她们看见直树以华丽的铲球夺走了相场的球，然后很快又起身朝球门进攻并且成功进球。

接着，现场响起一阵欢呼。

「成功了！高村同学好厉！！」

绫菜高兴得几乎要跳起来，脸上满是笑容。

「原本以为他只有脚程过人，这项才能还真教人意外呢。」

虽然红华有点佩服，但很快又皱起眉头。

「可是，为什么高村同学在踢足球呢？」

「我也不知道。」

绫菜摇摇头。

「是吗。」

红华一副受不了的表情，还发出『哼』的一声。

「明明川澄同学的情况那么严重，他还真是悠哉呢。我还以为高村同学是更重视朋友的人，看来我错看他了。」

直树其实就是在为了季代实奋斗，但不只是红华，就连身旁一脸困扰表情的绫菜也不可能知道这件事。

直树的挑战终于迈入最后关头。双方各得了两分，下一个进球的人就能获胜。

「没想到你居然这么有本事。」

把球放在中央的相场说着。尽管他脸上勉强还挂着笑容，但看起来极不自然。

「现在有这么多人在看，身为足球社社长，无论如何都不能输了。」

相场的眼神变得十分锐利。

「我要上了！」

他以非常凶猛的语气喊道，然后就一口气开球进攻。

「呀——！相场学长！就这样胜利吧！」

「不要输给那种货色～～！」

尖叫声四起，又是昨天那群自称粉丝团的女学生们。

「社长！请一定要赢！」

「替我们报仇啊！」

社员们也来了，他们正团结的为社长加油。

但就算如此，也不是没有人替直树加油。

「高村同学，加油！」

「高村，让他见识你的厉害！」

聚集而来的班上同学们，纷纷替直树喊声加油。

「高村学长！」

穿越人群跑到最前头的绘理子，把手圈在嘴巴前大喊着。

「绘理会全力替学长加油的！」

但直树并没有心力在意加油声。或者该说，他根本就没有注意到这么多的观众。

就算相场没有使用吸血鬼的力量，但他和身为人类的直树不同，不会感到疲惫。跑了将近十五分钟的直树已经十分疲倦，极限很快就要到来。

即使他追赶着正在盘球的相场，但两人之间的距离丝毫无法缩短。

（到此为止了吗……）

直树感觉到自己逐渐失去意识。虽然几乎要往前倒下，但他还是咬紧牙关。

（倒在这里像什么话！我要让川澄、让川澄得到幸福啊——！！）

他以几乎要失去感觉的双腿全速奔跑，追上了相场。以近乎神技的技巧抢走球后，转身朝反方向的球门狂奔。

「站住！！」

追在背后的相场脸上几乎让人感觉到怒气。他的双眼微微泛红，看来他已经施展出违反规则的吸血鬼之力。

相场以超越人类的速度进行移动，然后阻挡在直树面前。然而直树已经抵达射程。

「嘿呀啊啊啊！」

直树施展出最后的力气，将足球踢飞出去。球就这样飞过相场身旁，并且朝球门飞去。

经过一阵短暂的沉默，观众们发出热烈的欢呼。

「太厉害了！高村学长！」

绘理子跑到直树身旁，拉着他的右手蹦蹦跳跳的。

「没想到高村学长这么会踢足球……绘理好惊讶！而且好感动哟！」

替直树加油的班上同学们也带着笑容聚集而来。直到这时，直树才发现有这么多人在看着自己。

虽然他感觉有点高兴也有点不好意思，但现在不是沉浸在这种心情的时候了。

他走向呆站在原地的相场。

「相场学长，是我赢了。可以听听我的请求吗？」

11

「这件事情只有包含我在内的几个人知道，川澄现在生病了。」

直树刻意压低了音量。

这里是治水高中的校舍后方。大概是因为下午的课快要开始了，周围没有其他人在。

再怎么说，他们也不能在充满观众的校园里谈话，所以才会来到这个没有人的角落。

「那是一种名为吸血鬼昏睡症的疾病，生病的吸血鬼会昏睡不醒。治疗方法只有一个，就是在一星期内让异性吸血鬼亲吻。但是副作用会使那个人在醒来之后，迷上亲吻自己的人。」

一口气说明完后，直树注视着相场。

「相场学长，拜托你。能不能去吻川澄呢？」

「！？」

相场惊讶地睁大了双眼。

「我？为什么！？」

「因为川澄……川澄她……」

即使胸口感到一阵刺痛，但直树还是努力把话说出口。

「川澄她喜欢的人是相场学长。就算是为了痊愈，如果迷上不喜欢的男人就太可怜了。所以……」

「所以，你要我去唤醒她……？」

「是的。」

直树老实地点头。

「不勉强学长，如果你不喜欢川澄，或者已经有许下共度未来的女生了，那么我就放弃。可是如果不是那样……如果你能喜欢川澄的话，就求求你帮这个忙！」

相场默默地望着朝自己深深低头的直树，缓缓开了口：

「如果她迷上我，你可以接受吗？」

直树感觉内心有股难以言喻的感情，但他全力压抑下来。

「没关系……只要川澄能和喜欢的人在一起，我就心满意足了。」

「是吗，这就是你的愿望啊。」

相场动着嘴角，轻轻一笑。

「我明白了，接下来就交给我吧。」

「这么说……」

「没错，像她这么可爱的女孩子，我怎么可能有意见。幸好我现在没有特定的女朋友。你刚才已经赢我了，我会信守承诺的。」

「谢谢学长。川澄正在家里睡着。只要你向伯父伯母说是会长拍你去的，一定能到她的房间。」

相场把手放到直树肩膀上。

「你就期待明天的结果吧。」

留下这句话，相场就离开了校舍后方。

「事情告一段落了……吧。」

直树淡淡说道，他感觉到一阵强烈的疲劳感觉突然涌现。毕竟刚才在校园里全速奔跑，这也是当然的结果。

「这样一来，川澄就能让喜欢的相场学长亲吻并且醒来，两人永远在一起。」

直树想着，如果以童话故事来说，这应该是最好的结果。

但是……他内心却有一股超乎预期的不愉快。

（明明这是最好的解决办法，而且我也接受它了，为什么还会？）

就在直树自问自答的时候，一道声音响起。

「啊！直树！原来你跑到这里来了！」

下一秒钟，一名女学生从天上跳下来。眼前这名皮肤黝黑的活泼少女就是智衣。

「这不是智衣吗……」

直树抬头，发现三楼的窗户开着。

「我说，智衣。就算是抄近路，你也不该从三楼跳下来吧。要是有一天恢复了人类的身分，结果还改不掉这个习惯怎么办？后果可能不会只有骨折而已啊。」

智衣毫不理睬直树善意的规劝，就只是抓住他的肩膀用力摇晃。

「喂！直树！我听说了！你刚才在校园里大出风头啊！？他们说你彻底打败了足球社的相场！？」

智衣从后面抱住直树的头，还把他勒得紧紧的。

「为什么！你为什么没告诉我就跑去和他比拼！害我错过了一场好戏！」

看来她是因为没看到直树的活跃而感到气愤。

「所以，你要去哪啊？」

「去一下山田先生那里。」

在这个村里姓山田的只有一个人，那就是治水牧场的主人山田先生。

「咦，要到牧场去？」

直树有点惊讶。治水牧场位在郊外的山里，从这里过去不但要走路也要坐公车，有段不小的距离。并不是能在散步时顺便路过的地方。

「因为之前向他借了录影带。话说在前头，可不是色色的那种，是很健康的科幻电影。」

为了怕被季代实误会，直太郎特别强调着。

「因为已经借很久了，加上我也想借下一集来看。而且现在正在构思的科幻小说新作舞台也定在牧场，算是兼去取材吧。」

直树虽然一开始觉得牧场和科幻根本搭不上来，但马上就想通了。

说到牛，以一种被幽浮用光线吸进去的生物来说，算是非常有名的。直树之前不知道在哪里看过，在美国的一份统计中，凡是周围愈没有民宅的郊外牧场，就愈容易发生有人被外星人绑架的案件。

直树已经好久没有对父亲的工作产生兴趣，他开始基于好奇心发问。

「新作是什么内容啊？」

「新作可是足以进军世界舞台的科幻小说喔。从宇宙飞来的幽浮到了某个牧场，想用拖曳光线（Tractor Beam）抓走牛只时，不小心照到了刚好在那里的拖曳机（Tractor）上，结果就这样把拖曳机吸走了。用拖曳光线吸到拖曳机！这个同音笑话惹得外星人哄堂大笑。」

看来不是什么好内容。

「除此之外，我还想到一个故事，是说一个叫做※毕姆（Beam）的人曾吃过老虎，结果被幽浮拐走的故事。也就是说吃老虎的毕姆先生，被拖曳光线……」  （译注：「吃老虎的毕姆」与「拖曳光线」同音。）

还真的不是什么好东西。要是正常的编辑一定忍不住拿起扩音器大喊了吧。

喊着：『不采用』

而要是编辑先生够亲切的话，一定毫不犹豫地劝他换工作了吧。

就在直树整个人愣在那里的时候，家里的电话突然响了起来。

「来了、来了。」

为了接电话，直太郎走进房子里。

「那个，直树的父亲之前曾说过自己在写的是正统科幻小说……」

季代实说得很含蓄。

「他是不是弄错写作风格了？」

「毫无疑问就是错了。不，也许他能成为作家本身就已经大错特错了。」

直树一边叹气一边回答。

（说不定现在才劝他改行也不算晚。看是改行卖咖哩，还是专门替稀少部落翻译之类的都行。）

就在直树半认真地考虑这些事情的时候，直太郎又从家里走出来。

他身上穿的并不是工作服。虽然一样是松松垮垮的衣服，但至少是件西装。头发也特地用定型液整理成服贴的七三分线，夹在腋下的则是已经使用多年的旅行包。

这身行头代表着什么意思，直树马上就理解了。

「老爸，是出版社打来的电话吧，要你到东京去闭关写作之类的，我都警告你这么多次了，结果你还是拖稿了嘛。」

听到直树的话，直太郎皱起眉头加以反驳：

「说什么蠢话，我这次有遵守截稿期限。就只是因为原稿被作废了，要重写一遍而已。」

尽管他挺着胸膛自夸着，说的话却让人不敢领教。

「基于这个原因，家里就交给你看管了。啊，桌上那卷录影带，你在连续假期结束前随便挑一天，帮我拿去牧场还吧，替我向山田先生打声招呼。川澄同学，你可以慢慢坐。就算要住下来也没问题喔。」

直太郎朝着直树竖起大拇指。

「儿子，好好干啊！」

说完，直太郎就启程前往东京去了。

「什么叫做好好干啊。」

直树发出超大的叹息。

「抱歉，川澄，老爸又在胡说八道了。」

「不用在意，我玩得很开心。」

先是一笑，季代实又转向直树。

「对了直树，我有个小提议。」

「对，这个难得的假期，我想尝试一些与平常不同的事情。所以……」

季代实的话语，让直树的心跳几乎加速到极限。

（该、该不会是要说今晚想住下来吧？如果是的话我该怎么办：：）

他吞了口口水。

（父亲，我真的没有问题吗？）

不知为何，直树在内心以恭敬的语气询问着已经远去的父亲。

2

一辆公车停靠在治水村郊外的深山当中。在已经锈蚀的公车站牌上，写着『治水牧场前』的字样。

从公车上下来的人是直树与季代实。他们目送着公车离开后，又转头看向位在前方不远处的牧场入口。

「原来如此，你的提议是指这件事情啊。」

直树有点遗憾地低语着。

「我还以为……』

「还以为是什么？」

季代实抬头，一本正经地看着直树问道，当然直树无法诚实回答。

「没、没什么。我本来就想应该是来牧场玩。」

直树把手臂放到后脑袋附近，笑得十分虚假。

「刚好直树的父亲有事情找山田先生，偶尔在牧场悠闲地度过也不错吧？也许能够喝到现挤的牛奶也说不定。」

说完，季代实就朝牧场走去。

（川澄说的没有错，总比一直待在家里无所事事来得健康一点。而且，如果真的发生了『我以为』的事情，那才真的很严重。我不认为自己能够以平常心看待，要是太过亢奋而流了鼻血，搞不好会刺激到川澄的本能……）

光是想像就让直树背部发冷。

（以这点来说，到牧场约会也绝对不算坏事……）

想到这里，直树回过神来。

「不不，我怎么会说是约会呢？我和川澄又不是那种关系。」

看来一星期前的亲吻所带来的后遗症比想像中严重，无论如何直树都会太过意识到季代实的存在。

「直树，你在做什么？」

听到季代实的呼叫，直树才急忙走过去。两人就这样开始一起散步。

「对了，直树前阵子也来过这里呢。自以为是吸血鬼的时候。」

「拜托忘了那件事吧。」

直树露出极为苦涩的表情。

两个月前，直树误以为自己已经变成吸血鬼，还带着从东京前来的朋友们来到这个牧场。

不过，当时一群人马上就改去打『Mallet Golf』，只在牧场停留很短的时间。

「咦？」

季代实突然停下脚步，她正因为眼前出乎意料的景象而感到惊讶。

「还真冷清啊。」

直树嘟嚷了一句。

平常明明都有将近五十头的牛只在这里自由吃草，但今天却看不见任何一头牛。

「应该还不是赶回牛舍的时间吧。」

「是呀，现在应该是放牧的时间才对。」

由于直树曾经为了克服讨厌牛奶的习惯，而合季代实来这里帮忙过，两人对于这里的作息时间还很熟悉。

「该不会是有幽浮来这里，用拖曳光线把牛都劫走了吧？」

直树的想法的确很像是科幻小说家的儿子。

如果那是真的，伤脑筋的应该是外星人才对。

那些牛只已经被下过诅咒，生产的是能当作血液替代品的牛奶，当然不是普通的牛。它们晚上会倒吊在牛舍的天花板上睡觉，还会张开如同飞膜般的翅膀飞起来袭击人类。

光是想像到个头娇小的外星人被吸血牛袭击的景象，就让直树打从心底感到同情。

「别说傻话了，我们到牛舍去看看吧。也许只是时间和我们来帮忙的时候不太一样而已。」

季代实朝着牛舍走去，直树也跟着走。在离放牧场一段距离的地方，有一栋巨大的建筑物。这里是牛只晚上睡觉的牛舍，一阵牛只特有的酸味传了过来。

两人从巨大的入口处朝里头看，牛只还在。看来并不是被外星人拐走了。

只是，它们似乎都无精打采地横躺在干草铺成的地面上，还有许多工作人员正看着这些牛，忧心忡忡地讨论着牛只的状况，不过山田先生并不在这里。

「啊啊，到底该怎么办才好！？」

年轻的工作人员简直快哭了。

「够了，你还真吵耶！光是哎哎叫又能怎么样！」

上了年纪的工作人员难掩内心的焦躁，如此怒骂着。

「总而言之，这件事情绝对要保密，不可以让村子里的人知道。」

工作人员以严肃到令人害怕的表情，说出了这句话——

「牛、牛居然不再生产牛奶了。」

「什么——！」

发出惊叫的人是直树。即使季代实急忙捣住他的嘴巴，但为时已晚。工作人员已经转头看向这里，而且眼睛里还布满血丝。

季代实的手移开后，直树才手忙脚乱地开始解释：

「我、我们不是故意要偷听的。我只是来帮父亲还录影带给山田先生的，不是什么色色的东西，只是普通电影而已。」

由于现场气氛中甚至带有像是杀气的东西，使直树紧张到语无伦次的程度。

「那个，我不会把刚才听到的事情说出去的，请大家放心。不对，我可能根本没听清楚。牛怎么会不产牛奶了呢。」

看到直树还在继续说一些不说可能还比较好的藉口，季代实小声提醒他。

「直树，状况好像有点不妙。」

季代实说的没有错，工作人员们已经露出吸血鬼的眼神，还伸出尖牙利爪。

「我们快逃吧。」

季代实拉着直树的手，转身就跑。

「等等！」

「快追！」

两名工作人员开始追赶他们，一场追逐赛就这样开始了。

这里并不像学校社团活动那样，禁止使用吸血鬼的力量。因此他们正毫不留情地施展全力追逐。

「就算直树的脚程再快，这样下去都会被追上的。」

季代实的双眼开始变化成朱红色，她一把抓住跑在身旁的直树，并抱紧了他。虽说情况紧急，但对直树而言，被女孩子以公主抱的姿势抱着实在非常难堪。

虽然季代实打算一路冲下山逃走，但前方却有工作人员抄了近路意图阻挡。

「休想逃跑！」

即使季代实紧急转弯，但前方依旧有人阻挡着。

工作人员们包围住两人，而且还一边磨动着尖牙一边靠近过来。

「必须封口、必须封口。」

「总之，先捉起来关到仓库里吧。」

「不，应该挖洞埋起来比较好。」

「不对、不对，应该绑上重物然后丢进河里，几天内绝对浮不上来。」

（开开开、开什么玩笑！）

直树在内心大叫着。因为对方以为自己也是吸血鬼，所以说的事情非常偏激。

就在这个可称为绝境的时候，一道几乎能撼动树木的粗壮声音响了起来：

「你们这些笨蛋！」

飞奔过来的山田先生，开始把工作人员一一揍飞。由于他施展出吸血鬼的全力，所有人都飞得老远。

「可、可是山田先生。他们已经听到牛的事情了，一定要封口……」

「闭嘴～～！」

山田先生举起拳头朝最后一名工作人员头上敲下去，把他胸部以下的身体都敲入地面后，朝他们大吼。

「你们的眼睛长在哪里？他们看起来像是会随便泄漏秘密的人吗！？虽然只有几天，但他们之前明明那么认真帮我们牧场工作，不是吗！」

大概是吃了山田先生的一击后，脑袋冷静了不少，回到现场的工作人员们看起来平静了许多。

「的确，说要挖洞埋起来好像太夸张了一点。」

「我也觉得刚才说要把人丢进河里太可怕了。对方是吸血鬼也就算了，如果是人类不就是杀人了吗？」

（我完全是人类啊！）

直树在内心回答。

「小子、季代实，抱歉吓到你们了。因为事情太紧急，这些家伙都乱了手脚，就原谅他们吧。」

山田先生向两人深深低头致歉。

「请问，牛只真的不产牛奶了吗？」

面对季代实的询问，山田先生以苦涩的表情点头。

「是啊，很遗憾的，这是真的。今天早上我们原本想照平常一样地挤牛奶，但就是挤不出来。不论是花子、美子、留子、紫罗兰、俊江、麦代、阿静、阿龙还是伊莉莎白，就是没有一头牛挤得出牛奶。」

「以前有发生过这种事情吗？」

「没有，这还是第一次。所以才会束手无策啊！」

虽然山田先生重重地叹了口气，但他又勉强挤出笑容。

「不过，你们不用担心，这里的储乳槽还有牛奶。就算有几天不能挤牛奶，还不至于让村子缺粮。」

他像是想让两人放心。

「但是，你们也知道牛奶是村子里唯一的粮食。就算只是一时的现象，要是消息传开来将会造成其他村民的不安。所以拜托你们保密，答应我千万不要告诉任何人。」

山田先生直视着两人，现在无论如何都不能不答应。

直树与季代实用力点头。

「是吗，你们明白了吗，谢谢。」

山田先生以粗壮的手臂与两人握手后，又带着工作人员回到牧场去。

「虽然我说想要作一些与平常不同的事情，没想到会变成这样。」

季代实吐了口气，有点担心地看着牧场的方向。

「不过，真的没问题吗？不知道牛奶会不会恢复正常。」

「不必担心吧。那些牛就和这个村子的天空一样特别，是受过诅咒的。」

就因为直树亲身体验过牛只与众不同的地方，他的话语听起来特别有说服力。

「它们在晚上甚至能变身成怪物，当然要多少牛奶都没问题啊。山田先生也说过，一定很快就会恢复正常的。」

由于季代实的脸上还看得出忧心，直树轻笑着安慰她。

「不必这么担心，不会严重到用水稀释牛奶的程度吧。」

3

一转眼间。

黄金周连续假期就这么结束了。

包含六、日在内合计五天的假期，足以使人收不了心。

直树当然也不例外，即使到了午休时间，他还是不停地打着呵欠。

「什么啊，直树，怎么一直打呵欠。该不会在连续假期里熬了太多夜吧？」

向他攀谈的人，是刚从福利社回来的智衣。她的手里还握着两瓶牛奶。

「是啊，毕竟一个人在家，一不小心就会看电视或打电动到三更半夜。」

听到直树的感叹，智衣十分意外地询问。

「咦，一个人在家？你爸怎么了？」

「出版社打电话把他召集到东京去了。他到现在还没有回来，连通电话都没打。」

直树叹着气丢下这句话。

「八成又在几乎被监禁的情况下赶稿了吧。」

「科幻作家还真辛苦啊。居然得被别人关着写稿，那种事情我根本受不了。」

智衣打了个冷颤。对于不擅长写作文的她而言，想必是难以忍耐的状况。

但直树却不认为父亲值得同情，因为一切都是他自作自受的结果。

「哎，总之喝瓶牛奶提提神吧。」

智衣打起精神开朗地说完，把一瓶牛奶递给直树。

「我请客。」

「咦，真的吗？」

「是啊，不用客气，我现在还挺有钱的。」

智衣得意地说道。

「因为老爸说连续假期是赚钱的好时机，店里举办了特价活动，所有东西都打八折，所以这几天店里都挤满了客人。虽然整个假期我都在帮忙家里，不过也拿了很多打工的薪水。」

智衣打开牛奶，然后以口就瓶一口气干了它。

然而，喝完之后她的脸上却没有爽快，反而是一种难以言喻的表情。

「怎么了？」

「呃，该怎么说……我觉科味道好像比平常还淡。」

「没有那种事吧？」

直树也把智衣请的牛奶打开，喝了一口。喝惯的牛奶风味一下子在嘴里散开。但是，的确有不够满足的感觉。

如果在牛奶里加了冰块，喝起来大概就像这样吧？

「哎，应该是错觉吧。」

智衣笑着，她没有太在意，只是开玩笑地补了一句。

「山田大叔应该不会在牛奶里灌水吧。」

智衣这句话，一下子激发了直树脑袋中的记忆。他回想起一直忘记到现在的某件事情。

（该不会，牛只到现在还没有出乳吧？搞不好是因为储存的牛奶已经快用完，无可奈何下才灌水提供牛奶……）

「嗯，怎么了？直树，你怎么突然不说话了？」

「智衣，事情可能变得很严重了。」

直树一脸正经地向她说明。

「我在连续假期里，和川澄她……」

说到一半，直树连忙把话吞回喉咙。因为他想起来已经和山田先生约好，绝对不把这件事告诉任何人。

（要是不小心把秘密泄露出去，恐怕会吃上山田先生愤怒的铁拳啊。）

「你说在连续假期里和季代实怎样？这不是很教人在意吗？快说啊！」

面对焦急询问的智衣，直树全力在脸上挤出了一个假笑。

「没什么大事情，不用在意。谢谢你请的牛奶。」

直树把喝到一半的牛奶拿起来，向智衣道歉以后，就朝着季代实的座位移动。

「川澄，我有事情找你。」

「真巧，其实我也有话想说。」

季代实如此回答。

「该不会是跟黄金周的那件事情有关？」

「是呀，就是那件事。」

两人互相点了点头，然后就离开了教室，毫无疑问是打算进行不想被人得知的秘密谈话。

然后，有个人非常在意。

不用说，当然是被留下来的智衣。

「到底是什么事情？那件事情到底是什么？」

对于平常没什么耐心的她来说，要放着这个疑问不管是不可能的事情。

直树与季代实一路来到没什么人的校舍间通道角落，还慎重地确认了有没有其他人在。幸好，并没有看见会偷听别人说话的学生。

直树先是咽了一口口水，然后举起手里的牛奶问道。

「川澄，你喝过牛奶了吗？」

「有，刚刚才喝过。虽然说如果不是因为发生过连续假期那件事情，根本不会察觉到，但牛奶的确比较淡。看来直树的玩笑话成真了。」

季代实以手托住下巴思考着。

「会在牛奶里加水绝对不是小事。不知道是因为产量一直无法恢复……或者是更严重的，完全没有产量了。」

在沉重的气氛下，直树作了个提议。

「欸，川澄。放学后要不要到牧场一趟？我真的很在意耶。」

「是啊，就去看看吧。虽然问题不是我们去就可以解决的，但我也很在意。」

季代实点点头，然后提醒直树。

「不过这件事情不可以让其他人知道。毕竟牧场不是高中生会在放学以后去的地方。如果有人看到我们过去，一定会感到很不自然的。」

「嗯，是啊。必须避开别人的注意才行。」

稍作思考后，直树开了口。

「放学后先回家一趟，之后再到治水神社前碰面吧。那里应该不会有什么人。」

「也是，就这么作吧。」

两人迅速定了放学后的计画。

「那么，我们千万要注意别让其他人怀疑。就算是智衣也不可以乱说话。」

「嗯，我知道。虽然对智衣过意不去，但现在还得保密才行。」

当两人准备回教室时，季代实突然停下脚步。

「怎么了？」

「我觉得好像有人在。」

直树仔细观察四周，但外廊上并没有看见其他学生。他确认了放在远处的扫地用具柜，但里头只有扫把、水桶及拖把而已。

「是错觉吧，没有人在。」

「说的……也是，大概是我太神经质了吧。」

季代实也姑且接受，两人就这样离开了。

很遗憾的，那并不是错觉。

的确有人在偷听，就在窗户外。

像只壁虎般贴在通道外墙的，就是智衣。

「这下听到大消息了。」

智衣脸色苍白地说道。

顺带一提，智衣所指的『大消息』，并不是关于牛奶的事情。由于她来这里花了比预计更多的时间，只听见两人后半段的对话而已。

「居然会为了不让人知道而在神社见面，到底打算做什么？连我都不能知道，实在是太可疑了。」

「就是呀！这件事情真是太可疑了！可疑到不行呢！」

这道声音就在智衣身旁响起。仔细一看，一名头上绑着大蝴蝶结的女孩子也不分轩轾地和自己一样贴在墙上。

「绘理子！你为什么会在这里！？」

「因为绘理看见利根崎学姊贴在墙壁上，所以就忍不住好奇心了。别提这个了，重点是高村学长及川澄学姊。」

绘理子咬着大拇指的指甲。

「欵，绘理子，你觉得如何？能想像到他们两人在隐瞒什么吗？」

「从刚才两人之间的秘密谈话来看……」

绘理子的语气中充满不愿承认的心情。

「他们可能已经在交往了。」

「什么——！」

智衣贴在墙上，维持像壁虎般的姿态大叫着。

「再、再怎么说也不可能吧？」

「很难说哟，因为才刚发生过那件事情嘛。」

那件事情。智衣立刻明白是在说连续假期前那场吸血鬼的昏睡症骚动。

「该不会，直树就因为那个吻而对季代实……」

「川澄学姊可能也是因为高村学长救了自己而产生感情……而且也可以解释成多少还是有受到副作用的影响。」

「直树在教室里把话说到一半。他说连续假期里和季代实……该不会后面接的是『交往了』之类的吧。」

智衣用力摇头。

「不，那是不可能的，因为直树还是人类啊。如果已经在交往，就表示直树选了她吧？那么他应该已经被吸血，变成吸血鬼了。」

「该不会他们就是在谈这件事吧？说好今天放学以后，在神社见面……」

「然后，季代实就会吸直树的血……」

智衣的眼神中充满不甘心。

「我、我不接受！我绝对不会接受的！」

「绘理也不接受，高村学长的血是属于绘理的。」

「不不，稍等一下。为什么会是你的？直树的血应该给我吸才对。」

「那是什么时候决定的？在几月几日的几点几分几秒的时候！？」

虽然两人互瞪着，几乎爆出火花，但一下就察觉到这是毫无意义的争执。

「总而言之，还不确定就是这样吧。」

「是呀，现在最重要的应该是查明真相才对呢。」

两人互望了一眼，然后朝彼此点点头。

4

「嘿咻、嘿咻、嘿咻。」

直树一边喊着，一边面红耳赤地踩着脚踏车的踏板。由于通往牧场的道路是一连串的坡道，无论如何都需要这么大的力气。

「欸，直树。」

季代实的声音从后方传来。

「我还是下车吧。或者是我来骑车，直树坐在后面。」

目前的情况，是季代实侧身坐在脚踏车的后座。

「没、没问题，不用担心。这种程度的坡道不算什么。」

直树逞强地说着，使尽双腿的力气打算爬完没剩多少的坡道。

的确，季代实说得有理。身为吸血鬼的她，能以不输给脚踏车的速度奔驰。

相反的，如果是季代实来骑脚踏车并让直树坐在后面，想必他们早就抵达牧场了。

即使如此，直树依然很坚持这样的位置。

（因为上次让川澄以公主抱的姿势抱着我跑，这次一定要洗刷污名！）

「终、终于到了。」

直树像是攻上无人征服过的山顶一般，在成就感中下了脚踏车。虽然两脚都感觉到极度地疲劳，想必休息一下就会恢复了。

「没看见牛呢。」

季代实环顾着空荡的牧场低喃着。至于以山田先生为首的工作人员们，甚至连影子都看不见。

「欸，有没有听见什么声音？」

季代实突然问道。

「咦，没有啊？」

「不，的确有声音。这应该是莫札特的曲子。」

就在季代实发挥了吸血鬼独特的敏锐听力后。

「高村同学！？川澄同学！？」

一道尖锐的声音从后面响起。回头一看，红华就站在那里。她正以可怕的表情看着两人。

由于没有想到会在这里碰到红华，直树惊讶地瞪大双眼。

「会长！？你为什么会在这里？」

「我的事情不必多管。更重要的是，牧场现在很忙碌。你们在这里会碍事， 马上回去吧。」

红华强硬地开始赶人。

「我可以派司机开车送你们回家，总之快点……」

「会长，他们没关系的。」

绫菜慢慢跑过来，并且告诉红华。

「山田先生说过了，高村同学和季代实同学在第一天就知道了。所以也已经请他们保密了。」

「什么啊，原来已经知道了，害我这么紧张。」

看到红华呼了口气，直树问道。

「所以，会长为什么会来？」

「因为牧场主人山田先生，前天晚上有来找父亲商量。」

红华的家族——也就是神乐家——从很久以前就执掌了本村的权力。比较有年资的老居民一旦碰上问题，在前往村公所或村长那边之前，往往会先到红华家去徵求意见。

「父亲经过调查，已经得知牛只不产牛奶的原因了。所以，为了处理这件事情，我才会来这里坐镇指挥。」

红华转向绫菜。

「绫菜同学，真是不好意思，今天麻烦你请假来帮我。」

「不，没关系的，毕竟事态严重。」

绫菜严肃地答道。

「请问到底是为什么牛只不生产牛奶了？」

季代实问道。

「难道说是有人诅咒了牛只吗？」

虽然这问题很突兀，但并非不可能。这里的牛只本来就是受了诅咒，才会生产出能当作血液替代品的牛奶。

「并不是那么怪异的理由。」

红华缓缓摇头，然后把正确答案说出口。

「原因是压力。」

「压力？」

由于突然冒出很耳熟的名词，直树不禁傻住了。

「是的，就在牛只不能出乳的前一天，有一架直升机曾经在这座牧场的上空飞行。那是为了拍照制作村子的导览手册。据说村长先生亲自搭上直升机，在拍出满意的照片之前一直让直升机停留在上空。」

「意思是说，直升机的声音让牛只感到压力了？」

「对牛只而言，直升机螺旋桨的噪音似乎是难以忍受的痛苦。然后，它们就无法出乳了。」

就算如此，五十头牛突然全部都产不出牛奶，还是很离谱的事情。不过这座村子的牛是受了诅咒的特殊牛只，就算会有这种情况也不奇怪。

「村公所的人员还真会惹麻烦呢，就算不拍这里的照片也能制作手册吧。」

看到红华如此感叹，直树问了一个理所当然的问题：

「那个，会长。虽然这个问题可能无关紧要，不过为什么需要制作导览手册？这座村子不是尽量不引起注意比较好吗？」

「原则上是那样没错。不过，高村同学，难道你忘了吗？一年一度，会有一场祭典需要从外面找人来参加。」

红华的嘴角浮现带着某种含意的微笑。

「啊！」

直树的脑中浮现了不愿回想起来的讨厌记忆，那就是第一次体会到本村可怕之处的去年那场夕月祭。

（对了，我记得父亲以前确实有收到村子寄来的导览手册。）

光是想到今年又有新的牺牲品会被找来村子里，直树就感到一阵寒意。

「哎，再怎么抱怨也于事无补，现在替牛只消除压力才是最重要的。」

「可是，要怎么替牛只消除压力？」

看到直树不解地侧着头，红华得意地笑着。

「那么就特别让你看看吧。」

红华前往牛舍。在她拉开门进去之后，直树、季代实及绫菜都跟在后头。

牛只都在里头，它们正躺在比平常更舒服的干草床铺上。以山田先生为首的牧场工作人员们，正在以毛刷温柔地替牛只按摩。

牛舍里四处都放着喇叭音箱，并且有非常柔和的旋律从中播放出来。

音乐的源头是一台巨大的箱型唱片机。虽然看起来很有历史，但并不老旧，而是一般人也看得出来受过精心保养的高级品。

「这是从我家搬过来的唱片机。在播放的唱片则是一九七八年版的莫札特宗教音乐集，担任指挥的是被誉为名指挥家的罗宾·戈尼尔。」

红华轻轻闭上双眼，十分陶醉地欣赏着古典乐。

「如何？就算是不懂唱片的高村同学，应该也能明白这音乐的好吧？」

「嗯，我的确觉得很好听，也认为是好曲子，可是……」

可是，直树并没有勇气向充满自信的红华提出这个疑问。

「虽然还没有成效，但只要持续个两、三天，牛只们一定可以消除压力，像往常一样生产牛奶。」

红华看了牛只一眼，那是直树从来没有看过，一个充满不安的表情。

「真的万事拜托了，如果这样还不能产牛奶的话……到时候。」

不知道红华作了什么可怕的想像，她用力摇摇头。

「没问题的，一定能恢复生产。」

红华坚定地说道，像是要消除不安。

此时，牛舍外有一名工作人员探头进来。

「对不起，神乐大小姐。我们发现有两个人一直在偷看牛舍，我们把人抓起来了。」

被工作人员拉过来的不是别人，正是智衣和绘理子。看到这个情形，就连红华也无话可说。

「不、不是的。我们可没有什么奇怪的念头！」

「就、就是说啊，只是很好奇高村学长在里面做什么而已。」

「话说回来，红华，你在这里做什么？」

「大家为什么会一起来牧场，好奇怪哟。」

想要回避这个问题，是非常困难的一件事。

「真是拿你们没有办法。」

红华无奈地摇摇头。

5

「原来如此，是这么一回事啊。」

智衣正在位于牧场角落的员工铁皮宿舍中，恍然大悟地点着头。

红华等人已经将事情的来龙去脉说明清楚了。

「意思是说，高村学长和川澄学姊在连续假期里，偶然发现这件事，而且不能说出来是吗？然后因为福利社的牛奶很稀所以有点不安，两人才决定来看看情况。」

「因为事关重大，才会偷偷摸摸的吗。」

智衣和绘理子互看了一眼，似乎难掩兴奋之情。

「太好啦，看来不是最严重的事情。」

「就是呢，绘理真是松了一口气。」

尽管两人似乎发自内心地松了口气，但马上察觉到可怕的气氛。

红华舞动着长了利爪的手指，气愤地说道：

「现在明明处于我们最重要的食物——牛奶一滴也挤不出来的状况，你们居然还说太好了？」

「不、不是的，一点也不好。是吧，绘理子？」

「是、是呀，这样一点也不好呢。这是很严重的事情。」

就算伤口马上就会恢复，但是没有人想被红华狠狠一抓。

「不过红华，你的做法真的没问题吗？放那种音乐给牛听，它们根本听不懂吧？」

真不愧是智衣，她把直树不敢说的话全都说了出来。

「你对我的做法有意见，是吗？」

红华不悦地瞪着智衣。

「那么我问你，智衣同学，如果是你会怎么做？」

「说到消除压力当然就是要运动。把它们放到外面，然后拍拍屁股要它们跑一跑啊。」

听到智衣充满自信的意见，红华只是冷笑。

「如果交给智衣同学来办，我们就确定要饿死了。」

「什么啦，哪里有问题！？」

「全部都是问题！」

看到两人开始吵嘴，绫菜只能努力调解。

然后，绫菜用比较和缓的语气向智衣说明：

「让牛只听音乐，似乎真的有效哟。之前我在电视上有看过。如果让牛听音乐，它们的产量就会增加呢。听说那是一种音乐治疗。」

「这样啊，原来音乐这么厉害。」

智衣佩服地感叹着，但她的视线却停在绫菜的胸部上。就算以同性的立场来看，绫菜的胸部也十分令人赞叹。

智衣开了玩笑。

「绫菜，如果一直和牛一起听音乐，你是不是也会出乳啊？如果牛无论如何都不产牛奶，干脆你负责提供算了。」

「那、那是不可能的！」

满脸通红的绫菜遮着胸部喊道。

但直树的脸也一样红，不，应该是更红。

（不可以想像，不可以想像。一旦想像就输了。）

为了不做奇怪的幻想，直树不停地以咒语催眠着自己。

「总而言之，状况已经告诉你们了。应该不需要我提醒，但这件事情要是让村子里的人知道就糟了。不论是智衣同学还是绘理子同学，都不可以告诉任何人，知道了吗？」

「我知道啦。我没有笨到那种地步，不会跟别人说的。」

「绘理也没问题，已经很习惯保守秘密了。」

绘理子充满自信地挺着胸。

「秘密……啊啊，说得也是呢。」

红华轻轻一笑。

「仔细想想，我们这些成员已经保守秘密几乎一年了。让某人保持着人类之身，一个天大的秘密。」

「那个某人偶尔会惹出麻烦，让秘密差点穿帮就是了。」

季代实的发言让气氛缓和许多。

「真是的，为什么不能赶快下定决心呢，这样我们也能轻松点啊。」

「真的，绘理每天都有好好刷牙哟。」

在如此可怕的对话中，身为当事人的直树却完全没在听。

因为他还拼命地在内心念着咒语。

（不可以想像、不可以想像、不可以想……）

「哎，至少秘密还算守住了。我已经派人到邻镇去采购牛奶，从明天开始就会混到储备的牛奶里提供给村民。至少比加水更难被察觉。」

红华吐了口气，继续说道：

「如果能在不被任何人发现的情况下解决，是最理想的情形。毕竟要避免发生无谓的混乱。」

所有人都点头。没错，除了还在念咒语的直树以外。

「我说，高村同学，你到底有没有在听？」

大概是觉得他的态度不认真，红华吊起眼角。

（不可以想像、不可以想像、不可以想……）

「那个，高村同学。请问你还好吗？看起来好像不太舒服……」

绫菜担心地屈身去看直树，使得她的乳沟完全进入直树的视线当中。

这就是高村直树彻底败战的一刻。

他又变成许久不见的鼻血制造机。

「什、什么事也没有！！」

直树就这样按着鼻子，一边大叫一边冲出宿舍。

然后，来到隔天。

「呼，昨天真是受不了。」

直树如往常走在上学的路上，还如此自言自语着。

「都是因为智衣乱说话。就算泽田学姊的胸部比较大，怎么可以要她……」

由于差点又开始想像，直树使尽全力摇头。如果在这种地方变成鼻血制造机，事情可就严了。

（现在不要胡思乱想了。保守秘密，现在只要想这件事就够了。）

他又像是在念咒语般地重复想着这些事，然后换上室内鞋走在走廊上。就在这个时候——

「唔喔喔喔喔喔！！」

有名学生正发出这样的叫声，全力从走廊的另一头跑过来。那颗光头看起来很眼熟。

那人就是经常惹麻烦的同班同学熊谷。

「喂，熊谷。你在急什么啊？」

当熊谷正要经过直树面前时，直树喊住他。

以室内鞋急刹车的熊谷，一看到直树就连忙说道：

「当然要急啊！仔细听清楚了！吓一跳吧！牧场的牛已经挤不出牛奶了！」

熊谷的眼睛转动着，像是十分得意。

「什么——！」

直树的确吓到了，他惊讶的是熊谷居然会知道这件事情。

「不能再拖了！我要马上去福利社把所有牛奶都买下来！」

熊谷又开始奔跑，直树连忙追上他。

「等、等等啊，熊谷！」

直树打算问他在哪里听说这件事。另外，也打算阻止他继续传出去。

然而，当直树追到福利社时，他就发现自己必须放弃阻止熊谷了。

福利社已经聚集了大量的学生，所有人都杀红了眼抢购着牛奶。只要看了这个异常的景象一眼，就能知道他们已经得知村子里的紧急状况了。

「到底为什么秘密会被发现？」

智衣出现在惊讶不已的直树身后。

「直树，你吓到了吧？其实是因为一个意外的人泄漏了秘密。」

「意外的人？到底是谁？」

「就是山田大叔。」

智衣苦闷地皱着脸继续说道：

「昨天晚上，山田大叔有来我家的店里喝酒。大概是因为太过担心牛的状况，才不得不借酒浇愁吧。结果，他喝得比平常更醉。」

「该不会是因为喝得太醉，不小心把事情说出去了？」

「就是这样。喝醉之后大叔就开始哭，哭着说如果牛只再继续这样挤不出牛奶，身为牧场主人的他就对不起大家了。虽然我急忙把毛巾塞进他的嘴里，但为时已晚，所有来店里的客人都听到了。结果就变成这样。」

智衣耸耸肩。

「话说在先，不是只有这里，商店也是一早就挤满了人。」

「看来事情真的严重了。」

但此时的直树还不知道，事情还会变得更加严重许多。

6

&nbsp; &nbsp; 直树提着塑料袋，正在爬着校舍的楼梯。袋子里传来玻璃碰撞的响声，里头是他在福利社买来的五瓶牛奶。

&nbsp; &nbsp; 直树前往的是位在最高楼层的学生会室。直树像是回自己家一样自然，他没有敲门，直接就走进房间。

&nbsp; &nbsp; 智衣、绘理子以及绫菜都坐在学生会室里，她们纷纷以疲惫的神情转头看向直树。

&nbsp; &nbsp; 「咦？川澄呢？」

&nbsp; &nbsp; 「嗯？她没有和直树一起吗？」

&nbsp; &nbsp; 智衣露出有点讶异的表情。

&nbsp; &nbsp; 「算了，反正很快就会来吧。不管了，直树，赶快把牛奶拿来。」

&nbsp; &nbsp; 「喔，我知道了。」

&nbsp; &nbsp; 直树从塑胶袋里拿出牛奶，一瓶一瓶地拿给智衣等人。等到所有人都拿到后，彼此看了看对方。

&nbsp; &nbsp; 「好，那就开始喝啰。」

&nbsp; &nbsp; 大家打开瓶盖，开始喝着牛奶。

&nbsp; &nbsp; 「怎么样？我是喝不出来。」

&nbsp; &nbsp; 面对直树的询问，少女们无力地摇摇头。

&nbsp; &nbsp; 「今天也不行，这不是村子里的牛奶。」

&nbsp; &nbsp; 「这已经不是加什么东西的问题，百分之百是普通的牛奶呢。」

&nbsp; &nbsp; 「虽然不至于不好喝，也能填饱肚子。可是，无论喝多少都不能恢复力气。」

&nbsp; &nbsp; 绫菜悲伤地说着。

&nbsp; &nbsp; 她们所需要的是受了诅咒的特殊牛奶。无论喝再多普通的牛奶，也无法得到营养。

&nbsp; &nbsp; 「既然是普通的牛奶，就表示牛只还没有恢复过来，是吗？」

&nbsp; &nbsp; 直树看着喝了一半的牛奶。

&nbsp; &nbsp; 自从牛只不产牛奶以来，总共已经过了十天的时间。虽然之前是以普通牛奶混在里头，但如今就和绘理子说的一样，已经几乎都是普通的牛奶。

&nbsp; &nbsp; 光看她们的模样就能明白，她们一天比一天变得虚弱。虽然肉体上没有变化的吸血鬼不至于消瘦，但不只眼睛失去光彩，连说话及动作都变得有气无力。

&nbsp; &nbsp; 不是只有她们而已。其他学生，不，应该说所有村民都是这个状况。天空阴暗也是一个原因，整个村子的气氛都变得非常阴沉。

&nbsp; &nbsp; 「红华那家伙，白天都没来学校，应该都在牧场那边吧？难道都没有联络吗？」

&nbsp; &nbsp; 绫菜摇摇头，回答智衣的询问。

&nbsp; &nbsp; 「真的都没有联络。」

&nbsp; &nbsp; 「是吗？虽然很想去过看看情形……」

&nbsp; &nbsp; 智衣把整个头放在桌子上，无力地说道。

&nbsp; &nbsp; 「牧场好远啊。」

&nbsp; &nbsp; 平常喜欢运动的智衣居然会说这种话，让直树感到难以置信。

&nbsp; &nbsp; （意思是她们真的很缺乏营养吧。）

&nbsp; &nbsp; 由于只有自己正常用餐而保持着健康，这让直树觉得很过意不去。

&nbsp; &nbsp; 「欸，如果继续没有牛奶下去，我们会变得怎么样啊？」

&nbsp; &nbsp; 「还会怎么样呢，如果不能补充营养，一切都完蛋了。」

&nbsp; &nbsp; 「一定会变得无法动弹吧，然后就会一直沉睡下去。」

&nbsp; &nbsp; 难道营养不足会连思考也变得消极吗，三个人开始在像这样哀叹着。

&nbsp; &nbsp; 「没、没问题的！」

&nbsp; &nbsp; 为了改变阴沉的气氛，直树努力鼓励大家。

&nbsp; &nbsp; 「会长也在努力着，一定很快就有牛奶可以喝了！」

&nbsp; &nbsp; 但是，他自己最清楚这只是没有根据的鼓励而已。

&nbsp; &nbsp; 「对了，如果有什么事情需要我帮忙，尽管说没关系。什么事情都没问题。」

&nbsp; &nbsp; 直树为了替大家作点事情，决定如此提议。他认为这是唯一还有精神的人该尽的义务。

&nbsp; &nbsp; 「真的好吗？田径社的跨栏还没收好，我想要全都收回仓库里。还有，如果能帮我擦备用品就更好了。」

&nbsp; &nbsp; 「绘理的置物柜坏掉了呢，可以请高村学长帮忙修理吗？」

&nbsp; &nbsp; 「那个，我想拜托高村同学去教职员办公室回收学生会的预算档案，请问方便吗？另外还有花坛的保养。」

&nbsp; &nbsp; 大概是因为身体不听使唤，大家都有很多该做的事情没有完成。

&nbsp; &nbsp; （比想像中还要多啊。）

&nbsp; &nbsp; 虽然直树有点退缩，但他很快就要自己振作起来。

&nbsp; &nbsp; 现在只剩下自己还有精神，怎么可以不加油呢。

&nbsp; &nbsp; 「我知道了，小事一桩啦！」

&nbsp; &nbsp; 直树用力点头，并且向大家拍胸脯保证。

&nbsp; &nbsp; 直树手里抱着大量的档案夹，辛苦地爬着楼梯。这是绫菜托付的回收学生会预算档案的任务。

&nbsp; &nbsp; 接下的任务就要迅速执行！直树秉持着这个原则，立刻开始行动。

&nbsp; &nbsp; （也许像这种时候，就是展现男子气概的机会。）

&nbsp; &nbsp; 当直树沾沾自喜地搬着沉重的档案夹时，刚好遇见正在走廊上行走的季代实。

&nbsp; &nbsp; 「川澄，你怎么没有来学生会室？」

&nbsp; &nbsp; 「因为我有点不舒服，所以想到保健室休息一下。」

&nbsp; &nbsp; 季代实的脸色的确很差，看来营养不良所造成的憔悴程度远比智衣她们严重。

&nbsp; &nbsp; 「可是，保健室里已经挤满了休息的学生。所以我今天打算早退，已经和老师报备过了。」

&nbsp; &nbsp; 原来如此，难怪季代实的肩膀上挂着书包。

&nbsp; &nbsp; 「然后，直树在做什么？」

&nbsp; &nbsp; 被问到这件事，直树得意洋洋地开始说明。

&nbsp; &nbsp; 「智衣、绘理子还有泽田学姊都有托我办事情。因为我想替没有精神的大家作点事情，川澄有什么需要都尽管说吧。不管是要整理大布景，还是缝戏服都没问题。」

&nbsp; &nbsp; 「不用了，现在我这边没有事情。不过直树，你要注意不要太拼了。」

&nbsp; &nbsp; 季代实把脸凑近直树，小声地向他说道。

&nbsp; &nbsp; 「因为可能会有人注意到。」

&nbsp; &nbsp; 「啥？」

&nbsp; &nbsp; 季代实没有理会愣在原地的直树，就这样离开了。她的脚步看起来十分沉重。

&nbsp; &nbsp; 「太拼会有问题吗？」

&nbsp; &nbsp; 只留下感到不解的直树。

&nbsp; &nbsp; 如果是平常的季代实，一定会向他仔细说明原因吧，实际上也应该说清楚才对。

&nbsp; &nbsp; 但是她已经因为营养不足而全身无力，没有办法作到那个地步。

&nbsp; &nbsp; 烦恼了一阵子以后，直树自言自语道：

&nbsp; &nbsp; 「算了，应该不重要吧。」

&nbsp; &nbsp; 直树再次打起精神，继续向前走去。

&nbsp; &nbsp; 之后直树还是十分努力。

&nbsp; &nbsp; 他在下课时间，修理了绘理子的置物柜。

&nbsp; &nbsp; 放学后他立刻前往校园，完成智衣托付的工作。把丢在外面的跨栏收回仓库，也没有忘记保养备用品。

&nbsp; &nbsp; 接着轮到的是保养花坛。他依照绫菜的指示，把杂草拔除，赶走枝叶上的毛毛虫，然后浇上足够的水。

&nbsp; &nbsp; 等到结束所有工作之后，太阳已经开始下山了。

&nbsp; &nbsp; 「呼，终于结束了。」

&nbsp; &nbsp; 直树站在整理完毕的花坛前，满意地吐了口气。他满是汗水的脸上，正浮现出像是男子汉完成丰功伟业后的笑容。

&nbsp; &nbsp; 虽然直树不需要在一天之内全部作完，但他就是想替失去活力的她们作点事情。

&nbsp; &nbsp; 等到他转身准备回家时，发现身后站着智衣、绘理子及绫菜三个人。她们的表情上都充满了感激。

&nbsp; &nbsp; 「真没想到你一天就全部完成了，直树，谢谢你啦。」

&nbsp; &nbsp; 「置物柜能修好真是太好了，多亏高村学长的帮忙。」

&nbsp; &nbsp; 「高村同学，真的很谢谢你的帮忙。」

&nbsp; &nbsp; 三个人轮流向直树道谢。

&nbsp; &nbsp; 「看到直树这么努力，我也觉得自己不能就这样认输了。」

&nbsp; &nbsp; 「绘理也是。虽然身体还是没有什么力气，但内心似乎恢复精神了。」

&nbsp; &nbsp; 「就因为是这样的时候，意志才更应该振作起来才是呢。」

&nbsp; &nbsp; 看到自己的行动让三人稍微恢复了气力，直树真的感到非常高兴。

&nbsp; &nbsp; 此时，有几名男学生正在注视着直树。他们就站在能够俯瞰花坛的窗边，眼神还很空洞。

&nbsp; &nbsp; 「那是二年级的高村吧？还真勤劳啊。」

&nbsp; &nbsp; 「在动花坛之前，他好像还整理了田径社的备用品。」

&nbsp; &nbsp; 「我有看到他在午休时间搬一堆档案夹。」

&nbsp; &nbsp; 「还听说他在下课时间修理置物柜喔。」

&nbsp; &nbsp; 男学生们面面相觑，异口同声地提出疑问。

&nbsp; &nbsp; 「为什么他这么有精神？明明我们这么虚弱。」

&nbsp; &nbsp; 「已经好几天没有吃像样的东西了。不只是我们，全村的人都快虚脱了。」

&nbsp; &nbsp; 「没有吃饭的吸血鬼，怎么可能那么灵活地行动。」

&nbsp; &nbsp; 「啊！」

&nbsp; &nbsp; 其中一名男学生叫了一声。

&nbsp; &nbsp; 「记得那家伙的老爸还是人类吧？大家说要等他的大蒜味消除之后再吸血。该不会那家伙吸了他老爸的血？」

&nbsp; &nbsp; 「不，那是不可能的。」

&nbsp; &nbsp; 另一名男学生摇头否定。

&nbsp; &nbsp; 「就因为现在是这种状况，青年团的那些人早就不管什么大蒜味，想去吸他的血，不过，他根本就不在家。听说他去东京工作，一直没有回来。」

&nbsp; &nbsp; 「既然如此，他就没有吸他爸的血了。」

&nbsp; &nbsp; 「如果是这样，他之所以还能这么有精神……」

&nbsp; &nbsp; 男学生们原本空洞的眼神突然有了光亮。

&nbsp; &nbsp; 那是一种像是肉食动物找到猎物时的光亮。

&nbsp; &nbsp; 7

&nbsp; &nbsp; 「呼——这澡洗得真舒服。」

&nbsp; &nbsp; 身体冒着热气的直树，从浴室的更衣室里走了出来。然后从冷藏库拿出一瓶冰得恰好的牛奶。

&nbsp; &nbsp; 直树走到房子外面的狗屋前，将原本要喝的牛奶倒了一半在银色的容器里。

&nbsp; &nbsp; 「麻吕，来吃饭罗。」

&nbsp; &nbsp; 一只狗从狗屋里头慢慢地爬了出来。那是一只眼睛上的斑纹长得和眉毛一样，长相很有趣的狗。

&nbsp; &nbsp; 它正是高村家的爱犬·麻吕。

&nbsp; &nbsp; 麻吕将脸伸进容器里稍微舔了一下牛奶后，马上就不喝了，又回到狗屋里缩成一团。

&nbsp; &nbsp; 「果然不喝普通的牛奶吗？」

&nbsp; &nbsp; 直树感到很担心。

&nbsp; &nbsp; 高村家唯一一个变成吸血鬼的就是麻吕，理所当然地受到了这次粮食危机的影响。最近甚至严重到睡在狗屋里几乎不出来的程度。

&nbsp; &nbsp; 直树将剩下的牛奶喝完。喝起来感觉很好，对身为人类的直树来说在营养方面也没有什么问题。

&nbsp; &nbsp; 「说起来吸血鬼还真是不方便，居然不喝有诅咒的牛奶就活不下去。」

&nbsp; &nbsp; 直树若有所思地自言自语着，但又想到另一件事。

&nbsp; &nbsp; （不对，不是这样。其实没有牛奶也是可以生活的，只要吸血的话。）

&nbsp; &nbsp; 仔细想想牛奶只能算是本来作为粮食的血液之替代品。因为要是随意袭击人类，被发现的话就会发生很麻烦的事，所以应该说是村民们平常将就喝着被诅咒的牛奶才对。

&nbsp; &nbsp; 直树想起智衣之前曾经担心地说，要是牛只们一直产不出牛奶的话会怎么样？

&nbsp; &nbsp; 虽然智衣、绘理子或绫菜她们都想像过自己因为营养不足而倒下的模样，但其他村民们真会如此吗？

&nbsp; &nbsp; 为了生存下去而触犯禁忌的村人难道就不会多吗？

&nbsp; &nbsp; 触犯袭击人类的这个禁忌。

&nbsp; &nbsp; 直树不禁整个背脊发凉。

&nbsp; &nbsp; （也许闭关赶稿的时间拉长，对老爸来说反而是件好事。要是老爸现在在村子里，对村人来说可是上好的猎物。）

&nbsp; &nbsp; 此时，从玄关处传来了粗鲁的敲门声。

&nbsp; &nbsp; 「咦，是老爸！？」

&nbsp; &nbsp; 如果真的是他，这时机可说是再糟也不过，直树必须马上要他离开村子才行。

&nbsp; &nbsp; 直树从庭院入口处回到客厅，然后便走向玄关。或许是过度着急的缘故，他还没有确认站在门外的人是谁就直接打开了锁住的门。

&nbsp; &nbsp; 站在那里的并不是直太郎，而是和直树年纪相仿的男生们。

&nbsp; &nbsp; 虽然穿的是便服，但里头有几位直树很眼熟的人。他马上就察觉他们是治水高中的学长们。

&nbsp; &nbsp; 「不好意思，怎么了吗？」

&nbsp; &nbsp; 男学生们默默地走进房子里，然后把直树一路拉到客厅。

&nbsp; &nbsp; 虽然脸色因为营养不足的关系看起来不大好，但瞳孔仍持续发出异样的光芒。

&nbsp; &nbsp; （原来如此，他们肯定是为了吸老爸的血而来。）

&nbsp; &nbsp; 直树庆幸老爸逃过一劫，向众人说明着： 

&nbsp; &nbsp; 「那个，老爸他还在东京，现在……」

&nbsp; &nbsp; 「那种事情我们知道，我们要找的不是你爸。」

&nbsp; &nbsp; 其中一名男学生用手指着直树。

&nbsp; &nbsp; 「我们要找的就是你！高村直树！」

&nbsp; &nbsp; 「我？」

&nbsp; &nbsp; 「是啊，我们的眼睛可没有瞎掉。你今天在学校忙碌的样子我们全都看到了。」

&nbsp; &nbsp; 尽管学长们说得像是很有自信，但直树还是搞不清楚状况。

&nbsp; &nbsp; 「那有什么问题吗？」

&nbsp; &nbsp; 「问题可大了。你那样子怎么看都不像是好几天都没有正常吃饭的吸血鬼该有的样子！」

&nbsp; &nbsp; 直树叫了一声，他总算理解了眼前这些人想要表达的事情。

&nbsp; &nbsp; 在整间学校，不，在所有居民都因为营养不足而动弹不得时，直树还能像那样活蹦乱跳。

&nbsp; &nbsp; 作为村里的吸血鬼而言，直树的行动十分地不自然。

&nbsp; &nbsp; （就如川澄所说的，可能会有人注意到。）

&nbsp; &nbsp; 但是现在反省也已经来不及了。

&nbsp; &nbsp; 「我们已经知道你的秘密了。所以，我们是过来享用大餐的。」

&nbsp; &nbsp; 男学生们虽小尖牙笑着。

&nbsp; &nbsp; （就到此为止了吗……）

&nbsp; &nbsp; 其中一名男学生，朝着感到绝望的直树大喊：

&nbsp; &nbsp; 「高村，你是不是有私藏牛奶！？」

&nbsp; &nbsp; 「啥？」

&nbsp; &nbsp; 「我说牛奶啊，你一定偷藏了村里的牛奶，对吧？因为有那些牛奶可喝，所以你到现在还是很有精神，一定是这样没错！」

&nbsp; &nbsp; 看来他们似乎没有想到直树之所以能如此有精神，是因为他是个能够正常吃饭的人类。

&nbsp; &nbsp; （好、好险。看起来目标似乎不是我的血。）

&nbsp; &nbsp; 直树松了一口气，但要完全放心似乎还太早了。

&nbsp; &nbsp; 「把牛奶交出来！」

&nbsp; &nbsp; 「快给我拿出来！」

&nbsp; &nbsp; 男学生们不断地逼迫着，但高村家里根本没有可以给他们的牛奶。

&nbsp; &nbsp; 「当父亲从山田先生那里听说事情以后，立刻就决定使用那个方法。但我阻止了他，我问父亲能不能把这件事情交给我，用自己的方式处理。」

&nbsp; &nbsp; 「请问是为什么呢？如果有一个晚上就能解决的方法，为什么不马上那么做！？大家就不会这么痛苦了。」

&nbsp; &nbsp; 直树非常激动，话语中甚至带着责怪。

&nbsp; &nbsp; 「因为那个方法有问题。姑且不论父亲，但我无论如何都不能接受。没错，无论如何。」

&nbsp; &nbsp; 红华悲伤地垂下眼。她的表情一下子让直树恢复冷静。

&nbsp; &nbsp; （也对，既然没有翻译出来，就表示那方法应该有某些问题……）

&nbsp; &nbsp; 直树以百分之三百的严肃态度询问。

&nbsp; &nbsp; 「会长，请问那个方法是什么？」

&nbsp; &nbsp; 「是……」

&nbsp; &nbsp; 红华以苦涩的表情咬了咬唇，然后才静静答道。

&nbsp; &nbsp; 「让牛只们运动。」

&nbsp; &nbsp; 「啥！？」

&nbsp; &nbsp; 直树的严肃态度一下子就归零了。

&nbsp; &nbsp; 「你说运动，就是动动身体的那个运动？智衣最喜欢的。」

&nbsp; &nbsp; 「是的，就是那个运动。只要让牛只全力运动，消除压力后就能再度出乳了。」

&nbsp; &nbsp; 红华认真地点头。

&nbsp; &nbsp; 「什么啊，原来是运动，只要那样就可以了吗？」

&nbsp; &nbsp; 直树先是发愣，然后又开朗地说道。

&nbsp; &nbsp; 「既然这样不就更该由智衣出马？只要像她在牧场时说的那样，拿棒子去打牛的屁股，逼它们跑一跑……」

&nbsp; &nbsp; 「我原本也这么想，在去年我得知村子的牛的真面目之前。」

&nbsp; &nbsp; 直树的回想起可怕的记忆。

&nbsp; &nbsp; 为了克服讨厌牛奶的习惯，直树在牧场住了一晚。但他却在牛舍看见一群宛如蝙蝠般倒吊着睡觉的牛，那些怪物还曾经拍动着像是飞膜般的脚当作翅膀，追杀着身为人类的直树。

&nbsp; &nbsp; 直树感觉到他已经克服的惧牛症又快复发了。

&nbsp; &nbsp; 「父亲告诉我。想让牛只全力运动，就必须让它们以夜晚的吸血牛姿态尽情飞舞。而且，它们就只有在追人的时候才会露出本性。意思是说，如果想要再次得到牛奶，就必须准备活人当作祭品才行。为什么我会用祭品来形容，你明白理由吗？」

&nbsp; &nbsp; 直树静静点头。

&nbsp; &nbsp; 被追逐过的直树十分明白。被那些吸血牛袭击的话，肯定不会只有被吸血而已。一定会被它们啃个精光的。

&nbsp; &nbsp; 「原来会长不想把人类当作活祭品，所以才想透过其他方法解决……」

&nbsp; &nbsp; 他觉得刚才批评红华的自己很丢脸。

&nbsp; &nbsp; 「虽然夕月祭里总是追着人类的我，可能没资格说这种话。但至少夕月祭不会死人。就算被袭击而吸血，也只会变成吸血鬼而已。可是，这次却不一样。」

&nbsp; &nbsp; 红华以双臂抱着自己的身体。

&nbsp; &nbsp; 「如果为了我们吸血鬼的生存，牺牲了人类的性命……我觉得我们届时将会变成真正的怪物。所以，就算比较花时间，我也想藉由其他方法来解决。虽然还是失败了。」

&nbsp; &nbsp; 红华怨叹着自己的无力，像是死心地叹了口气。

&nbsp; &nbsp; 「父亲说的没有错，村子已经来到极限了。为了解救村子，一定要请别人牺牲才行。」

&nbsp; &nbsp; 像是在坚定悲痛的决心，红华紧咬着嘴唇。

&nbsp; &nbsp; 「明天傍晚我会到邻镇去。要把几个人带回村子里来。」

&nbsp; &nbsp; 「咦，会长要亲自去吗！？」

&nbsp; &nbsp; 红华朝惊讶的直树静静点头。

&nbsp; &nbsp; 「那就是父亲先前让我处理的条件。如果我失败的话，就要帮助父亲去收集活祭品。」

&nbsp; &nbsp; 红华的确非常适合。以她的美貌，随便搭讪都会有男人傻呼呼地跟上来。

&nbsp; &nbsp; 就连眼前等待他的就是地狱也不知道。

&nbsp; &nbsp; 「我真是卑鄙呢，居然和高村同学说这些话。」

&nbsp; &nbsp; 红华细语着。

&nbsp; &nbsp; 「这座村子将会牺牲人类性命而得救。如此残酷的事实，我居然选了高村同学和自己一起背负。明明高村同学只是不小心听到，要怎么敷衍过去都无所谓的……」

&nbsp; &nbsp; 面对直树的红华，脸上一点也看不出平常骄傲的模样，她的双眼正浮现大粒的泪珠。

&nbsp; &nbsp; 「对不起，真的很对不起。」

&nbsp; &nbsp; 红华以微弱的声音道歉，然后就宛如逃跑一般地跑走了。

&nbsp; &nbsp; 「会长……」

&nbsp; &nbsp; 直树就只能注视着她的背影而已。

&nbsp; &nbsp; 9

&nbsp; &nbsp; 直树推着脚踏车，一个人踏上归途。

&nbsp; &nbsp; 他感到十分郁闷。

&nbsp; &nbsp; 「没想到让牛出乳的方法，居然会是那样。」

&nbsp; &nbsp; 他回忆起红华最后的哭脸。平常既骄傲又有气质的红华，居然会让直树看见那样的表情。

&nbsp; &nbsp; 以平常总是觊觎其血液而言，直树明明一直被她压在脚下。

&nbsp; &nbsp; 光是这样，就能看出她承受了多少压力。

&nbsp; &nbsp; 明天晚上，将有人类被当成祭品献给吸血牛。

&nbsp; &nbsp; 然后，负责去诱惑那些人的，正是红华的工作。

&nbsp; &nbsp; 对方是吸血牛，想要让它们好好运动绝对不可能只靠一个人。三人、六人，搞不好会需要更多。

&nbsp; &nbsp; 那些人数将会成为红华背负的十字架。

&nbsp; &nbsp; 直树心想，这实在太过分了。

&nbsp; &nbsp; 他并不希望让红华做那种事，也讨厌以活祭献牛这种解决法。对同样身为人类的直树来说绝不能接受。

&nbsp; &nbsp; 可是，令人悲伤的是，事实上就只剩下这个方法能解救村子。

&nbsp; &nbsp; （也不是一定要让吸血牛吃人吧，只要让它们好好运动就够了。只要逃个一晚，牛只应该就能出乳了。）

&nbsp; &nbsp; 虽然如此，但直树也很明白那几乎不可能办到。

&nbsp; &nbsp; 毕竟那些可是吸血牛，就因为直树曾被追杀过，才比别人明白它们有多么快速。

&nbsp; &nbsp; 没错，包含足以冻僵背脊的可怕吼叫声，以及如同飞膜般的翅膀能刮起多强的烈风，这些直树全都知道。

&nbsp; &nbsp; 「………………」

&nbsp; &nbsp; 直树停下脚步，他的脸上挂着沉重的表情。

&nbsp; &nbsp; 「最好的结局，就是让身为活祭的人活下来。如此一来就能再度挤出牛乳，会长也不须背负任何罪孽。」

&nbsp; &nbsp; 直树愈说愈大声。

&nbsp; &nbsp; 「最好由脚程够快，又很清楚敌人的人物来担任活祭品。」

&nbsp; &nbsp; 直树只知道一个人符合条件。

&nbsp; &nbsp; 没错，那个人就是他自己。

&nbsp; &nbsp; 直树深呼吸，然后用力点头。

&nbsp; &nbsp; 这就是他下定决心的一刻。

&nbsp; &nbsp; 他跳上脚踏车，并且用力踩下踏板，朝着目的地开始全速冲刺。

&nbsp; &nbsp; 「我现在已经比那时候更了解地形了，所以不必怕找不到路逃跑！」

&nbsp; &nbsp; 为了激励自己，直树刻意大喊。

&nbsp; &nbsp; 「我现在已经比那时候更有体力！因为已经被追杀过好多次了！」

&nbsp; &nbsp; 声音愈来愈大。

&nbsp; &nbsp; 「我现在已经比那时候更喜欢这座村子了！不管是川澄、智衣、绘理子、泽田学姊，或者是会长，我都想守护她们！这一次，是我为了守护村子和她们的战斗。所以我不能输！」

&nbsp; &nbsp; 最后，直树竭尽全力喊道：

&nbsp; &nbsp; 「给我等着吧！你们这五十头大奶妹！看我把你们整得惨兮兮！」

&nbsp; &nbsp; 早晨造访了治水牧场。

&nbsp; &nbsp; 山田先生跌跌撞撞地从员工宿舍走出来。

&nbsp; &nbsp; 他的脸上看不到任何光彩。不只是营养不良，他的精神也很憔悴。

&nbsp; &nbsp; 因为他一直承受着身为牧场主人的责任压力。

&nbsp; &nbsp; 「反正今天也挤不出牛奶吧。」

&nbsp; &nbsp; 过去的豪爽气度已经荡然无存，变得极为悲观的山田先生如此感叹着。

&nbsp; &nbsp; 「明天也是、后天也是，都不会有牛奶。」

&nbsp; &nbsp; 他喃喃自语着，并且打开牛舍的门。

&nbsp; &nbsp; 山田先生踏入牛舍。牛只们已经起床了，并且站起来动着嘴巴。

&nbsp; &nbsp; 「然后，治水村就要这样灭亡了。已经没救……」

&nbsp; &nbsp; 山田先生的话只说到一半。他瞪大眼睛，嘴巴张得像是可以放入拳头一般。

&nbsp; &nbsp; 因为他察觉到了，牛只的奶头前方正在滴着白色液体。

&nbsp; &nbsp; 而且不是只有一头牛。不论是花子、美子、留子、紫罗兰、俊江、麦代、阿静、阿龙还是伊莉莎白，放眼望去每一头牛都在滴着奶水。

&nbsp; &nbsp; 山田先生马上奔出牛舍，朝着懒散走出宿舍的员工们发出怒吼。

&nbsp; &nbsp; 他的表情甚至看不出来到底是在哭，还是在笑。

&nbsp; &nbsp; 「马上准备Milk er！！女儿们的大奶子正在哭啦！」

&nbsp; &nbsp; 随着牛只们的复活，山田先生的低级黄色笑话似乎也像凤凰一般重生了。

&nbsp; &nbsp; 10

&nbsp; &nbsp; 在满是乌云的天空下，智衣正以轻快的脚步跑着。

&nbsp; &nbsp; 她身上已经完全看不出几天前的沉重。以田径社的优美姿势奔跑的她，脸上正带着清爽无比的笑容。

&nbsp; &nbsp; 她的手里握的并不是接力棒，而是一瓶牛奶。

&nbsp; &nbsp; 智衣来到郊外的一栋房屋，这里是直树家。

&nbsp; &nbsp; 「嗨！直树！我进来罗！」

&nbsp; &nbsp; 智衣毫不客气地打开玄关的门，然后就脱了鞋子跳进屋内。

&nbsp; &nbsp; 「起床了吗！？直树！？有大新闻啊！」

&nbsp; &nbsp; 以明朗的声音大声喊着并踏人客厅的智衣，发现脚边躺在令她意外的人物。

&nbsp; &nbsp; 「季、季代实！你为什么会睡在这里！？」

&nbsp; &nbsp; 季代实无力地抬起脸。

&nbsp; &nbsp; 「咦，已经天亮了吗？结果我好像在这里过夜了呢。」

&nbsp; &nbsp; 「过过过、过夜了！你你你怎么可以这样！现在直树他老爸不在家，意思是昨天晚上只有你和直树……」

&nbsp; &nbsp; 大吃一惊的智衣，手里的牛奶瓶掉到了地上。掉在杨杨米上的牛奶瓶就这样滚到季代实面前。

&nbsp; &nbsp; 季代实随手捡起牛奶，发现瓶子意外地温。

&nbsp; &nbsp; 「这个怎么不冰？」

&nbsp; &nbsp; 「咦，喔喔，当然啊。因为这是现挤的牛奶，我从牧场那里急忙带过来的。因为我想到麻吕大概很饿才拿来的。话说季代实，你刚才还没有回答我……」

&nbsp; &nbsp; 但季代实并没有仔细听智衣的询问，她光是听到『现挤的牛奶』，就马上打开瓶盖喝。

&nbsp; &nbsp; 一口气把牛奶喝光的季代实，眼神恢复了原来的精神。

&nbsp; &nbsp; 「好喝，果然村子里的牛奶就是不一样。」

&nbsp; &nbsp; 季代实陶醉地吐了口气后，转头看向智衣。

&nbsp; &nbsp; 「牛奶、牛奶已经恢复生产了吗！？」

&nbsp; &nbsp; 「是啊，之前的停产简直像是幻觉一样，现在正在大量生产着。整个村子欢欣鼓舞，所有人都在狂喝送来的牛奶呢。不对，那些事情不重要了。」

&nbsp; &nbsp; 智衣露出可怕的表情。

&nbsp; &nbsp; 「季代实，你要说明清楚。为什么你会住在直树家！？昨昨昨、昨天晚上到底出了什么事？」

&nbsp; &nbsp; 「放心，没有发生智衣担心的事情。」

&nbsp; &nbsp; 喝完牛奶，恢复平常从容态度的季代实如此答道。

&nbsp; &nbsp; 「我来看直树的情况以后，就无力地昏倒了，一直睡到刚才。」

&nbsp; &nbsp; 即使得到说明，智衣还是不太能接受的样子。

&nbsp; &nbsp; 此时，高村家里又响起另一道声音。

&nbsp; &nbsp; 「高村学长！！有大新闻了！牧场的牛只们又开始产牛奶了！」

&nbsp; &nbsp; 冲进家里的是绘理子。她手里也和智衣一样紧握着牛奶。

&nbsp; &nbsp; 接着，绫菜也来了。

&nbsp; &nbsp; 「高村同学！牛奶、牛奶又恢复生产了！」

&nbsp; &nbsp; 熟面孔们纷纷集合在一起。

&nbsp; &nbsp; 「绘理子、绫菜，你们一大早就来干嘛？」

&nbsp; &nbsp; 智衣不高兴地问道。

&nbsp; &nbsp; 「当然是来报告呀。因为高村学长也很担心，绘理想赶快让他知道。也想让他看看喝完牛奶变得精神百倍的绘理。啊，还有带牛奶来给小麻吕喝。」

&nbsp; &nbsp; 「我也是，想到小麻吕可能在饿着肚子。」

&nbsp; &nbsp; 汪汪！

&nbsp; &nbsp; 后院传来麻吕的吠叫声，看来它已经感觉到有营养的牛奶来了。

&nbsp; &nbsp; 绘理子与绫菜走到后院，并且把牛奶倒在麻吕的碟子上。麻吕马上发出唏哩呼噜的声音全喝光了。

&nbsp; &nbsp; 满意地看着麻吕喝完后，绘理子又转身问道：

&nbsp; &nbsp; 「所以，高村学长在哪里呢？」

&nbsp; &nbsp; 「啊，对了。我还没有碰到直树耶。」

&nbsp; &nbsp; 由于被季代实过夜的发言所动摇，智衣完全忘了这件事。

&nbsp; &nbsp; 「喂——直树！你在哪里？」

&nbsp; &nbsp; 尽管智衣在狭窄的房子里呼叫直树，却没有得到回应。

&nbsp; &nbsp; 「那家伙是不是还在睡啊？明明是这么可喜可贺的时候。」

&nbsp; &nbsp; 智衣奔上楼梯，但很快地又闷着表情下楼。

&nbsp; &nbsp; 「奇怪，高村那家伙不在家耶。」

&nbsp; &nbsp; 「对了，刚才好像没有看见高村学长的鞋子。是不是出门了呢？」

&nbsp; &nbsp; 当所有人都百思不解时，季代实突然想到。

&nbsp; &nbsp; 「直树会不会是从昨晚到现在都没有回来？」

&nbsp; &nbsp; 「你知道些什么？」

&nbsp; &nbsp; 三人纷纷询问季代实。

&nbsp; &nbsp; 「昨天晚上，直树说要去找会长询问牛只的事情，然后就奔出门了。也许他还没有回来。」

&nbsp; &nbsp; 「他到红华家去了？」

&nbsp; &nbsp; 智衣的表情凝重。

&nbsp; &nbsp; 「昨天晚上还没有产牛奶，红华那家伙一定也很饿了。要是直树就这样大剌剌地出门……」

&nbsp; &nbsp; 「高村学长！搞不好已经被神乐学姊吃掉了！？」

&nbsp; &nbsp; 「那是不可能的，会长不是那么没有节操的人……」

&nbsp; &nbsp; 虽然绫菜打算替红华说话……

&nbsp; &nbsp; 「可是毕竟状况特殊……也许……」

&nbsp; &nbsp; 「赶、赶快去红华她家！」

&nbsp; &nbsp; 「是！」

&nbsp; &nbsp; 智衣和绘理子从玄关跑出去，绫菜和季代实也追在后头。

&nbsp; &nbsp; 然而，她们并没有抵达红华家，因为她们很快就碰见想要寻找的人物。

&nbsp; &nbsp; 通往高村家的路途中，直树正拿木棒当作拐杖，拖着脚步缓缓前进着。他的衣服变得破破烂烂，全身也沾满泥土，还有许多擦伤或割伤。

&nbsp; &nbsp; 这么适合以全身是伤来形容的年轻人还真不多见。

&nbsp; &nbsp; 「我……撑过一个晚上了。」

&nbsp; &nbsp; 话语中所带的疲惫，让他感觉到自己好像快吐出血来了。

&nbsp; &nbsp; 虽然和宣示的不同，被整惨的是直树，但总之他还是努力逃了一整晚。

&nbsp; &nbsp; 那真的是非常严酷的一晚。为了躲避吸血牛们的猛攻，直树跑遍了整座森林。有时候甚至还跳进水里躲避。差点死掉的次数，就算叫全班同学拿双手手指来数也不够用。

&nbsp; &nbsp; 像这样幸存下来的直树，绝对可以断定一件事。如果再发生同样的事情，自己毫无疑问会死。

&nbsp; &nbsp; 这次的生还就是如此靠运气。

&nbsp; &nbsp; 「直树！」

&nbsp; &nbsp; 「高村学长！」

&nbsp; &nbsp; 「高村同学！」

&nbsp; &nbsp; 前方传来声音，使直树抬起脸来。他发现对面出现的是智衣、绘理子、绫菜，以及季代实。

&nbsp;&nbsp;「既然大家都恢复精神了，意思是牛奶已经恢复生产了吗？」

&nbsp; &nbsp; 直树并没有剩下任何力气能够到牧场去确认。但光是看到智衣她们跑过来的模样，就能得知牛只已经恢复出乳了。

&nbsp; &nbsp; 「太好了，真是……太好了。」

&nbsp; &nbsp; 直树感到非常欣慰。跑过来的智衣急忙问道：

&nbsp; &nbsp; 「喂，直树！你到底是怎么搞的！？为什么看起来这么凄惨？」

&nbsp; &nbsp; 「难道是被神乐学姊袭击了吗？所以才在逃跑途中变成这样……」

&nbsp; &nbsp; 「不是那样的，其实是……」

&nbsp; &nbsp; 虽然直树打算把自己的丰功伟业拿来炫耀，但……

&nbsp; &nbsp; （如果要提吸血牛的事情，就必须提到会长的事情啊。）

&nbsp; &nbsp; 包括红华为了拯救村子，决定牺牲人类的事。

&nbsp; &nbsp; 「其实……」

&nbsp; &nbsp; 直树把手放到头的后方，嘿嘿地笑着。

&nbsp; &nbsp; 「我从会长家回来的时候不小心迷路，结果脚一滑就从山上摔到河里，真是有够惨的。」

&nbsp; &nbsp; 「真是的，你就是这么粗心。」

&nbsp; &nbsp; 「就是呀，晚上走路请小心一点。」

&nbsp; &nbsp; 一辆高级轿车在不远处停了下来。从后座下车的红华，看见全身破破烂烂的直树。

&nbsp; &nbsp; 「我就在想，果然是你做的吗，高村同学。」

&nbsp; &nbsp; 红华的表情像是在哭，也像是在笑。

&nbsp; &nbsp; 「明明死了也不奇怪，反而是活下来才稀奇的行为，但你却真的去做了。而且，解救了整旧村子。」

&nbsp; &nbsp; 红华又摇摇头。

&nbsp; &nbsp; 「不对，不只是村子，还包括我。」

&nbsp; &nbsp; 她以湿润的双眼注视着直树。

&nbsp; &nbsp; 「高村……同学。」

&nbsp; &nbsp; 「啊，对了。直树你听了以后别吓到喔！牛只又开始产牛奶了耶！」

&nbsp; &nbsp; 智衣像是刚刚想起，马上向直树报告。

&nbsp; &nbsp; 「喔～～～真的啊，那太好了。」

&nbsp; &nbsp; 直树的演技真是烂透了。

&nbsp; &nbsp; 「什么嘛，怎么反应那么冷淡。我们之前可是面临生死关头耶。」

&nbsp; &nbsp; 智衣不满地嘟着嘴唇，然后朝直树施展了一记颈部关节技。

&nbsp; &nbsp; 「如何，有牛奶可喝之后我就恢复力气啦，厉害吧？」

&nbsp; &nbsp; 她用力地勒着直树的头。

&nbsp; &nbsp; 「我、我知道了，知道了啦，快点放开我。」

&nbsp; &nbsp; 就在直树发出哀号的时候。

&nbsp; &nbsp; 「啊，会长！」

&nbsp; &nbsp; 绫菜喊了一声。

&nbsp; &nbsp; 「喔，这不是红华吗！」

&nbsp; &nbsp; 智衣放开了直树，朝着红华用力挥手。

&nbsp; &nbsp; 「还以为会怎样呢，不过你还是让牛奶恢复生产啦。嗯，真有一套。真不愧是我们的学生会长啊。」

&nbsp; &nbsp; 「就是啊！都是神乐学姐的功劳！」

&nbsp; &nbsp; 但红华像是没有听见智衣与绘理子的赞美声，一路走到直树面前。

&nbsp; &nbsp; 红华的脸颊微微泛红，还以异常湿润的性感眼神注视着直树。

&nbsp; &nbsp; 她以像是吐气般的甜美声音，呼唤着直树。

&nbsp; &nbsp; 「高村……同学。」

&nbsp; &nbsp; 「是。」

&nbsp; &nbsp; 「高村……同学。」

&nbsp; &nbsp; 「是。」

&nbsp; &nbsp; 「高村……同学。」

&nbsp; &nbsp; 「……是。」

&nbsp; &nbsp; 就在直树伤脑筋地作出第三次回答后，红华就伸出了手，用力抱住直树。直树因为过度疲劳而不小心向前倾，导致头的位置必然会冲向红华的胸口。

&nbsp; &nbsp; 红华抱着直树的头，大声地喊道：

&nbsp; &nbsp; 「高村同学！我爱你！」

&nbsp; &nbsp; 「什么——————！」

&nbsp; &nbsp; 「什么——————！」

&nbsp; &nbsp; 「什么——————！」

&nbsp; &nbsp; 智衣、绘理子以及绫菜都以不输给红华的声音惊叫着。虽然没有喊出声音，但季代实也惊讶地睁大双眼。

&nbsp; &nbsp; 「红、红华！你在干什么！快点离开直树！」

&nbsp; &nbsp; 尽管智衣扑过去想把两人分开，但红华却抱着直树，华丽地闪避开来。

&nbsp; &nbsp; 「能不能请你不要来妨碍，这是属于我和高村同学的甜蜜时刻。」

&nbsp; &nbsp; 「会、会长，你是在开玩笑吗？」

&nbsp; &nbsp; 绫菜的声音听得出来像是在说『请说你是在开玩笑』，然而……

&nbsp; &nbsp; 「我怎么可能在开玩笑。高村同学不只救了村子，同时也是我的恩人。我会喜欢上他是理所当然的。」

&nbsp; &nbsp; 红华露出幸福的笑容答道。

&nbsp; &nbsp; 「我原本就很中意高村同学了，也早就决定要吸高村同学的血。不过，我已经改变想法了。血给谁都无所谓，只要高村同学的心属于我的就好。」

&nbsp; &nbsp; 红华又把直树抱得更紧了。

&nbsp; &nbsp; 至于直树，已经接近昏迷了。虽然论起胸部大小是绫菜胜出，但红华傲人的身材依旧不输给模特儿。

&nbsp; &nbsp; 现在可是被她夹在胸部之间，他怎么可能保持住正常的精神状态。

&nbsp; &nbsp; 「好了，高村同学来我家吧，我会温柔地替你包扎的。」

&nbsp; &nbsp; 红华带着直树，以像是跳舞般的步伐打算离开。

&nbsp; &nbsp; 「站住！红华！我不会把直树交给你的！」

&nbsp; &nbsp; 「就算是神乐学姊，绘理也不会输的！」

&nbsp; &nbsp; 智衣与绘理子追在后头。

&nbsp; &nbsp; 「唉，如果会长成为对手，我怎么可能赢得了呢。可是，可是我……」

&nbsp; &nbsp; 绫菜先是自言自语着，然后又下定决心向前奔跑。

&nbsp; &nbsp; 「事情好像变得很惊人呢。」

&nbsp; &nbsp; 被留在原地的季代实，虽然慢了一拍，依然感到十分惊讶。

&nbsp; &nbsp; 「直树，一定又做了什么惊人之举，是足以让会长动心的大事情。」

&nbsp; &nbsp; 季代实笑着，因为她以直树为荣。

&nbsp; &nbsp; 「我也不能输呢。」

&nbsp; &nbsp; 血吸村阴暗的天空下，季代实也开始奔跑。
